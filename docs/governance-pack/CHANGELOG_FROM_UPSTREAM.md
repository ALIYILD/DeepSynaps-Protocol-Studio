# Changelog from Upstream

This file documents every removal and replacement made during the sanitization lift
from `ALIYILD/[upstream-repo]` to the DeepSynaps governance pack.

Upstream-specific strings are represented here in redacted form using the tags below
to prevent re-introduction of prohibited terms into this pack:

- `[UP-BRAND]` = upstream product brand name (the repository name root)
- `[UP-MODULE]` = upstream Python module name
- `[UP-CLI]` = upstream CLI command prefix
- `[UP-KNO]` = upstream knowledge base path prefix
- `[UP-INST]` = upstream institution name
- `[UP-PROP-NET]` = upstream proprietary network assessment acronym
- `[UP-PROP-FIELD]` = upstream proprietary schema field name

Format: `FILE | LINE (approx.) | TYPE | ORIGINAL (redacted) | REPLACEMENT / ACTION`

## Files NOT Lifted

The following upstream files were intentionally excluded:

- Any YAML or architecture document describing the proprietary multi-network assessment
  architecture (e.g. network definitions YAML, bedside assessment templates)
- Any document whose primary purpose is to implement or describe the proprietary
  6-network bedside assessment instrument

These files were excluded because they depend on proprietary architecture not applicable
to DeepSynaps Studio and because they are framed around the prohibited assessment
framework.
