# DeepSweeper Scope

DeepSweeper is limited to conservative maintainer cleanup for DeepSynaps GitHub
issues and pull requests. It may review open items, write review artifacts, and
support human maintainers with closure suggestions.

It is out of scope for DeepSweeper to make clinical decisions, modify patient
data, bypass regulated-component review, or auto-close items that touch
regulated paths, protocol logic, privacy controls, or compliance workflows.
