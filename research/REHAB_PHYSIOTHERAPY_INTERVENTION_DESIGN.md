## 1. Rehabilitation Domains

### 1.1 Neurological Rehabilitation

Neurological rehabilitation addresses impairments resulting from damage to the central or peripheral nervous system. The field has evolved from compensatory approaches to embrace neuroplasticity-driven recovery principles.

#### 1.1.1 Stroke Recovery

**Epidemiology & Clinical Context:**
- Worldwide, 15 million people suffer stroke annually; 5 million are left permanently disabled
- Leading cause of long-term adult disability globally
- Recovery trajectory: most rapid in first 3 months, continues beyond 6-12 months with appropriate intervention

**Impairment Domains:**
- Motor function (hemiparesis/hemiplegia)
- Spasticity and muscle tone abnormalities
- Sensory deficits (proprioception, stereognosis)
- Cognitive deficits (attention, memory, executive function)
- Perceptual deficits (unilateral neglect, anosognosia)
- Aphasia and communication disorders
- Dysphagia
- Emotional and psychosocial challenges

**Evidence-Based Interventions:**

| Intervention | Evidence Level | Target Outcomes | Key Parameters |
|---|---|---|---|
| Repetitive Task Practice (RTP) | Strong (A) | Motor function, ADL | 300-600 reps/session, 5x/week |
| Constraint-Induced Movement Therapy (CIMT) | Strong (A) | Upper extremity function | 2-week intensive, 6h/day constraint |
| Body-Weight Supported Treadmill Training (BWSTT) | Moderate (B) | Gait, ambulation | 20-30 min, 3x/week, 30% BWS |
| Virtual Reality Training | Moderate (B) | Motor learning, engagement | 30-45 min sessions, game-based |
| Mirror Therapy | Moderate (B) | Motor function, neglect | 30 min, 5x/week |
| Robotics-Assisted Training | Moderate (B) | Upper/lower extremity | Device-specific protocols |
| Neuromuscular Electrical Stimulation (NMES) | Moderate (B) | Wrist/finger extension | 30-60 min, 5x/week |
| Mental Practice / Motor Imagery | Moderate (B) | Motor function | 15-20 min, combined with physical |
| Action Observation Therapy | Emerging (C) | Motor learning | Video observation + execution |
| Bilateral Arm Training | Moderate (B) | Coordination, function | 45-60 min, 3x/week |

**Neuroplasticity Principles Guiding Intervention:**

1. **Use it or lose it:** Neural circuits not actively engaged weaken over time
2. **Use it and improve it:** Training drives specific neural changes
3. **Specificity matters:** The nature of training determines the nature of plasticity
4. **Repetition matters:** Sufficient repetition is required to induce lasting changes
5. **Intensity matters:** Higher training intensity produces greater gains
6. **Time matters:** Different forms of plasticity occur at different times post-injury
7. **Salience matters:** Training must be meaningful and attention-capturing
8. **Age matters:** Younger brains show greater plasticity, but adult neuroplasticity persists
9. **Transference:** Plasticity in response to one training experience can enhance acquisition of similar behaviors
10. **Interference:** Plasticity from one experience can interfere with acquisition of other behaviors

**The EXCITE Trial Framework for CIMT:**
- Inclusion: Some active wrist extension and finger extension
- Protocol: Restrain unaffected limb 90% waking hours for 2 weeks
- Intensive training: 6 hours/day of structured task practice
- Shaping methodology: Gradual increase in task difficulty
- Transfer package: Home practice, problem-solving strategies
- Expected outcomes: Significant improvement in WMFT (Wolf Motor Function Test)

**Post-Stroke Gait Training Progression:**

```
Phase 1 (0-2 weeks):   Bed mobility → Sitting balance → Standing tolerance
Phase 2 (2-4 weeks):   Stand-pivot transfers → Pre-gait activities → Weight shifting
Phase 3 (4-8 weeks):   Parallel bars → Walker → Single-point cane
Phase 4 (8-12 weeks):  Cane → Unsupervised ambulation → Community ambulation
Phase 5 (12+ weeks):   Advanced gait (stairs, uneven terrain) → Running progression
```

#### 1.1.2 Traumatic Brain Injury (TBI)

**Classification:**
- Mild TBI (GCS 13-15): Concussion, brief loss of consciousness (<30 min)
- Moderate TBI (GCS 9-12): Loss of consciousness 30 min - 24 hours
- Severe TBI (GCS 3-8): Loss of consciousness >24 hours

**Rehabilitation Domains:**
- Physical: Balance, coordination, strength, endurance, headaches
- Cognitive: Attention, processing speed, memory, executive function
- Behavioral: Emotional regulation, irritability, impulsivity, motivation
- Communication: Expressive/receptive language, pragmatics
- Sensory: Visual, vestibular, auditory processing

**Evidence-Based Interventions:**

| Domain | Intervention | Evidence Level |
|---|---|---|
| Physical | Task-oriented gait training | Strong (A) |
| Physical | Balance training (progressive) | Strong (A) |
| Cognitive | Cognitive rehabilitation (compensatory) | Moderate (B) |
| Cognitive | Computerized cognitive training | Moderate (B) |
| Cognitive | Dual-task training | Moderate (B) |
| Behavioral | Cognitive-behavioral therapy | Strong (A) |
| Vestibular | Vestibular rehabilitation | Strong (A) |
| Fatigue | Energy conservation + graded exercise | Moderate (B) |
| Sleep | CBT-I + light exposure | Moderate (B) |

**Concussion-Specific Protocol (Progressive Return to Activity):**

```
Stage 1: Complete rest (24-48 hours post-injury) - cognitive and physical
Stage 2: Light aerobic activity (walking, swimming) - HR < 70% max, no resistance
Stage 3: Sport-specific exercise (skating, running) - add movement, no head impact
Stage 4: Non-contact training drills - progressive resistance, cognitive load added
Stage 5: Full-contact practice - restore confidence, assess functional skills
Stage 6: Return to play - medical clearance required
```

*Progression criteria: 24-hour symptom-free period at each stage before advancing.*

#### 1.1.3 Spinal Cord Injury (SCI)

**Classification:**
- Complete vs. Incomplete (ASIA Impairment Scale A-E)
- Tetraplegia (cervical) vs. Paraplegia (thoracic/lumbar)
- Neurological level of injury determines functional expectations

**ASIA Impairment Scale:**

| Grade | Description | Functional Implications |
|---|---|---|
| A | Complete: No motor/sensory function preserved in S4-S5 | Wheelchair dependent |
| B | Incomplete: Sensory but no motor function below neurological level | Limited motor recovery |
| C | Incomplete: Motor function below level; <50% key muscles with grade 3 | Potential for ambulation |
| D | Incomplete: Motor function below level; >=50% key muscles with grade 3 | High ambulation potential |
| E | Normal: Motor and sensory function normal | Fully functional |

**Key Interventions:**
- Locomotor training (body-weight supported treadmill)
- Functional electrical stimulation (FES) cycling
- Robot-assisted gait training
- Activity-based therapy (high-repetition, task-specific)
- Strength training for innervated muscles
- Respiratory muscle training (especially for cervical injuries)
- Autonomic dysreflexia prevention and management
- Pressure injury prevention

#### 1.1.4 Multiple Sclerosis (MS)

**Disease Course Types:**
- Relapsing-Remitting (RRMS): 85% of initial diagnoses
- Secondary Progressive (SPMS)
- Primary Progressive (PPMS)
- Progressive-Relapsing (PRMS)

**Rehabilitation Targets:**
- Fatigue management (primary intervention priority)
- Balance and ataxia
- Spasticity management
- Weakness and deconditioning
- Gait dysfunction
- Bladder/bowel dysfunction
- Heat sensitivity (Uhthoff's phenomenon)

**Intervention Considerations:**
- Exercise does not worsen disease progression
- Submaximal exercise protocols preferred
- Aquatic therapy for heat-sensitive patients
- Energy conservation strategies essential
- Cooling garments for exercise tolerance
- Resistance training: moderate loads, avoid maximal exertion
- Interval training may be better tolerated than continuous

**Evidence-Based Exercise Prescription for MS:**

| Modality | Parameters | Frequency | Evidence |
|---|---|---|---|
| Aerobic | 50-70% HRmax, 10-30 min | 2-5x/week | Strong |
| Resistance | 50-80% 1RM, 8-15 reps | 2-3x/week | Moderate |
| Balance | Static/dynamic, progressive | 2-3x/week | Moderate |
| Aquatic | 27-29 degrees C pool | 2-3x/week | Moderate |

#### 1.1.5 Parkinson's Disease

**Cardinal Symptoms:**
- TRAP symptoms: Tremor at rest, Rigidity, Akinesia/bradykinesia, Postural Instability
- Secondary: Gait disturbance (shuffling, festination, freezing), cognitive decline, depression, autonomic dysfunction

**Exercise as Neuroprotective/N eurorestorative:**
Growing evidence suggests exercise may have disease-modifying effects through:
- Neurotrophic factor upregulation (BDNF, GDNF)
- Neuroplasticity enhancement
- Improved dopaminergic transmission
- Reduced neuroinflammation

**Lee Silverman Voice Treatment (LSVT) BIG Protocol:**
- High-amplitude movement training
- 16 sessions over 4 weeks (4x/week, 60 min each)
- 5 daily homework exercises (10 min each)
- Focus on perceived normal amplitude (patient tends to underscale)
- Carryover tasks into daily activities

**PD-Specific Exercise Framework:**

```
Intensity: HIGH (70-85% HRmax) - evidence supports vigorous exercise
Type: Progressive resistance + aerobic + balance + flexibility
Frequency: 3-5x/week
Duration: 30-60 min
Key components:
  - Amplitude-based training (LSVT BIG principles)
  - Dual-task training (cognitive + motor)
  - Cueing strategies (visual, auditory, tactile)
  - Forced exercise (motorized cycling, tandem training)
  - Complex movement sequences
  - Functional movement patterns
```

**Cueing Strategies:**
- Visual: Transverse lines on floor, laser canes
- Auditory: Metronome, rhythmic auditory stimulation (RAS)
- Tactile: Vibratory cues, sensory cues
- Cognitive: Attentional strategies, motor imagery

**Disease Stage-Specific Adaptations:**

| Stage (H&Y) | Exercise Focus | Intensity | Precautions |
|---|---|---|---|
| 1-2 (Unilateral/Bilateral) | High-intensity, amplitude | Vigorous | Monitor orthostatic BP |
| 3 (Mild-moderate bilateral) | Balance, gait, dual-task | Moderate-vigorous | Fall precautions essential |
| 4-5 (Severe) | Seated, supported, functional | Light-moderate | Adaptive equipment, caregiver |

### 1.2 Musculoskeletal Rehabilitation

Musculoskeletal rehabilitation encompasses the largest portion of physiotherapy practice, addressing conditions affecting bones, joints, muscles, ligaments, and tendons.

#### 1.2.1 Low Back Pain

**Classification Framework:**
- **Specific back pain:** Underlying pathology identified (fracture, malignancy, infection, cauda equina)
- **Non-specific low back pain (NSLBP):** 90% of cases; no identifiable structural cause
- **Referred/radicular pain:** Nerve root involvement

**Risk Stratification (STarT Back Tool):**

| Risk Level | Characteristics | Intervention Approach |
|---|---|---|
| Low | Few psychosocial factors | Minimal intervention, self-management |
| Medium | Physical + some psychosocial | Physiotherapy (physical + reassurance) |
| High | Significant psychosocial barriers | Combined physical + psychological approach |

**Classification-Based Treatment (Treatment-Based Classification System):**

1. **Stabilization subgroup:** Movement control deficits, hypermobility
2. **Extension-oriented subgroup:** Respond to repeated extension
3. **Flexion-oriented subgroup:** Spinal stenosis, spondylolisthesis
4. **Traction subgroup:** Signs of nerve root compression
5. **Immobilization/pain modulation:** Acute, severe pain

**Exercise Prescription by Subclassification:**

| Subgroup | Primary Exercises | Secondary | Parameters |
|---|---|---|---|
| Stabilization | Transverse abdominis, multifidus, pelvic floor | Global core, hip strengthening | Hold 10s, 10 reps, daily |
| Extension | Prone press-ups, extension in lying | Prone lying, standing extension | 10 reps, every 2 hours |
| Flexion | Knee-to-chest, seated flexion | Flexion-biased walking | 10 reps, 3-4x/day |
| Traction | Mechanical/manual traction | Nerve glides, positioning | Per protocol |

**Core Stability Training Progression:**

```
Phase 1: Motor control training
  - Abdominal hollowing (TA isolation)
  - Multifidus activation (prone, co-contraction with TA)
  - Neutral spine awareness
  - Breathing integration

Phase 2: Static stabilization
  - Dead bug (supine, alternating limb movement)
  - Bird dog (quadruped, contralateral extension)
  - Plank (forearm, progressive duration)
  - Side plank

Phase 3: Dynamic stabilization
  - Stability ball exercises
  - Cable/resistance perturbations
  - Functional movement patterns with core control

Phase 4: Functional integration
  - Sport-specific movements
  - Compound lifts with proper mechanics
  - Work simulation activities
```

#### 1.2.2 Shoulder Rehabilitation

**Common Pathologies:**
- Rotator cuff tears (partial/full-thickness)
- Subacromial impingement syndrome
- Adhesive capsulitis (frozen shoulder)
- Glenohumeral instability
- Labral tears (SLAP, Bankart)
- Shoulder arthroplasty (reverse/total)

**Rotator Cuff Repair Rehabilitation (Phased):**

```
Phase 1: Protection (0-6 weeks)
- Sling immobilization (6 weeks)
- Passive ROM only (patient/therapist-assisted)
- Scapular stabilization (isometrics)
- Modalities for pain management
- Goals: Protect repair, prevent stiffness, manage pain

Phase 2: Active ROM (6-12 weeks)
- Discontinue sling
- Active-assisted → active ROM
- Begin light resistance (1-2 lbs)
- Scapular strengthening
- Proprioceptive training
- Goals: Full passive ROM, initiate active motion, scapular control

Phase 3: Strengthening (12-20 weeks)
- Progressive resistance for rotator cuff
- Closed-chain exercises (wall push-ups)
- Open-chain strengthening
- Functional movement patterns
- Goals: Restore strength, improve function

Phase 4: Return to activity (20-26 weeks)
- Plyometrics (if required)
- Sport/work-specific training
- Gradual return to full activity
- Goals: Full functional return, symptom-free activity
```

**Adhesive Capsulitis (Frozen Shoulder) Protocol:**

| Stage | Clinical Features | Treatment Focus | Modalities |
|---|---|---|---|
| Freezing (1-3 months) | Painful, progressive stiffness | Pain management, gentle ROM | Corticosteroid injection, TENS, gentle mobilization |
| Frozen (4-12 months) | Stiffness predominant | Aggressive stretching, mobilization | Joint mobilization (Maitland/Kaltenborn), end-range loading |
| Thawing (12-24 months) | Gradual improvement | Strengthening, function restoration | Progressive resistance, functional training |

#### 1.2.3 Knee Rehabilitation

**Common Pathologies:**
- ACL/MCL/LCL/PCL tears
- Meniscal tears
- Patellofemoral pain syndrome
- Osteoarthritis
- Total knee arthroplasty (TKA)

**ACL Reconstruction Rehabilitation Overview:**
See detailed protocol in Section 4 (Protocol Templates).

**Patellofemoral Pain Syndrome (PFPS) - Evidence-Based Approach:**

**Contributing Factors:**
- Quadriceps weakness (VMO specifically)
- Hip abductor/external rotator weakness
- Tightness: IT band, hamstrings, quadriceps, calves
- Excessive foot pronation
- Overuse/training errors

**Exercise Hierarchy (Evidence Strength):**

1. **Hip strengthening (strongest evidence):**
   - Hip abduction (side-lying clamshells)
   - Hip external rotation
   - Single-leg Romanian deadlift
   - Monster walks with resistance band

2. **Quadriceps strengthening:**
   - Terminal knee extension with band
   - Short-arc quads
   - Leg press (pain-free ROM)
   - Step-downs (progressive height)

3. **Neuromuscular training:**
   - Single-leg balance (eyes open/closed)
   - Star excursion balance test training
   - Perturbation training

4. **Patellar taping:**
   - McConnell taping for immediate pain reduction
   - Facilitates exercise tolerance

#### 1.2.4 Hip Rehabilitation

**Common Pathologies:**
- Femoroacetabular impingement (FAI)
- Labral tears
- Hip osteoarthritis
- Total hip arthroplasty (THA)
- Greater trochanteric pain syndrome (GTPS)

**THA Rehabilitation Protocol (Posterior Approach):**

```
Phase 1: Protection (0-2 weeks)
- Hip precautions: No flexion >90 degrees, no adduction past midline, no internal rotation
- Weight-bearing: As tolerated (per surgeon)
- Gait training with assistive device
- Ankle pumps, quad sets, glute sets
- Supine exercises only

Phase 2: Early strengthening (2-6 weeks)
- Progress weight-bearing
- Wean from assistive device
- Bridging (if cleared for flexion)
- Standing hip abduction, extension
- Stationary bike (upright when cleared)

Phase 3: Functional strengthening (6-12 weeks)
- Progressive resistance training
- Single-leg exercises
- Step-ups/downs
- Balance training
- Pool therapy (if incision healed)

Phase 4: Advanced (12+ weeks)
- Return to recreational activities
- Sport-specific training (if applicable)
- Plyometrics (if appropriate)
- Discharge with home program
```

#### 1.2.5 Sports Injuries

**Common Sports Injuries & Rehabilitation:**

| Injury | Acute Phase (0-7d) | Subacute (1-6w) | Remodeling (6w-6mo) | Return to Sport |
|---|---|---|---|---|
| Ankle sprain | POLICE, early mobilization | Proprioception, strengthening | Functional training, sport-specific | Hop test, Star Excursion |
| Hamstring strain | POLICE, isometric holds | Progressive loading (Nordic curls) | Running progression, agility | Askling H-test, sprint mechanics |
| ACL reconstruction | See Section 4 | See Section 4 | See Section 4 | >9 months, criteria-based |
| Shoulder dislocation | Sling, pain management | ROM, rotator cuff strengthening | Proprioception, sport-specific | Closed kinetic chain tests |
| Achilles tendinopathy | Load reduction | Heavy slow resistance (HSR) | Energy storage loading | Plyometric progression |
| Groin strain | POLICE, gentle ROM | Adductor strengthening | Running, cutting drills | Pain-free sport simulation |

**Tendinopathy Management - Updated Evidence:**

The pathological model has shifted from inflammation to failed healing response:

| Stage | Characteristics | Exercise Approach |
|---|---|---|
| Reactive | Acute overload, swelling | Relative rest, isometrics (pain-monitoring) |
| Tendon dysrepair | Attempted healing, matrix breakdown | Progressive loading, heavy slow resistance |
| Degenerative | Cell changes, matrix disorganization | Heavy slow resistance, energy storage loading |

**Heavy Slow Resistance (HSR) Protocol:**
- 3 sets of 15 repetitions at 70% of 1RM
- Progress to 3 sets of 12 at 75%, then 3 sets of 10 at 80%
- Slow tempo: 3 seconds concentric, 3 seconds eccentric
- 3x/week for 12 weeks minimum
- Key: Pain during exercise acceptable if settles within 24 hours

### 1.3 Cardiopulmonary Rehabilitation

#### 1.3.1 Cardiac Rehabilitation

**Phases of Cardiac Rehabilitation:**

| Phase | Timing | Setting | Focus |
|---|---|---|---|
| Phase I (Acute) | In-hospital | Hospital | Early mobilization, risk stratification, education |
| Phase II (Subacute) | 2 weeks - 6 months | Outpatient center | Structured exercise, risk factor modification |
| Phase III (Maintenance) | 6+ months | Community/home | Long-term maintenance, lifestyle |
| Phase IV | Ongoing | Community | Independent exercise, peer support |

**Indications for Cardiac Rehabilitation:**
- Myocardial infarction (STEMI, NSTEMI)
- Coronary artery bypass graft (CABG)
- Percutaneous coronary intervention (PCI/stenting)
- Stable angina
- Heart failure (stable, chronic)
- Heart valve repair/replacement
- Cardiac transplant
- Peripheral artery disease

**Exercise Training Components:**

1. **Aerobic training:**
   - Mode: Treadmill, cycle ergometer, rowing
   - Intensity: 40-80% HRR or 50-85% VO2 reserve
   - Initial: 15-20 min/session
   - Progression: Up to 60 min/session
   - Frequency: 3-5x/week

2. **Resistance training:**
   - Initiated 2-5 weeks post-MI or post-surgery (per physician clearance)
   - Intensity: 30-70% 1RM
   - 1-3 sets, 10-15 repetitions
   - 8-10 exercises covering major muscle groups
   - Avoid Valsalva maneuver (exhale on exertion)

3. **Breathing exercises:**
   - Diaphragmatic breathing
   - Pursed-lip breathing
   - Thoracic expansion exercises

**Risk Stratification for Exercise Prescription (AACVPR):**

| Risk Level | Characteristics | Supervision | Intensity |
|---|---|---|---|
| Low | No complications, preserved LVEF >50% | Minimal/Education | 70-85% HRmax |
| Moderate | Moderate LV dysfunction (EF 40-49%), controlled symptoms | Intermittent supervision | 60-75% HRmax |
| High | LVEF <40%, complex arrhythmias, heart failure symptoms | Continuous telemetry | 40-70% HRmax |

#### 1.3.2 Pulmonary Rehabilitation

**Indications:**
- Chronic obstructive pulmonary disease (COPD) - primary indication
- Interstitial lung disease
- Bronchiectasis
- Cystic fibrosis
- Pre/post lung transplant
- Post-COVID syndrome
- Lung cancer (pre/post surgical)

**Core Components:**

| Component | Details | Frequency |
|---|---|---|
| Exercise training | Lower extremity aerobic, resistance | 2-3x/week, 6-12 weeks |
| Upper extremity training | Arm ergometry, unsupported arm exercises | Integrated into sessions |
| Education | Disease management, medications | Weekly sessions |
| Nutritional counseling | Weight management, optimization | As needed |
| Psychosocial support | Anxiety, depression, support groups | Ongoing |
| Self-management | Action plans, exacerbation management | Integrated |

**Exercise Prescription for COPD:**

| Modality | Parameters | Notes |
|---|---|---|
| Walking/cycling | 20-60 min, 60-80% peak work rate | Interval or continuous |
| Interval training | 30-60s high intensity, 30-60s rest | Better tolerated in severe COPD |
| Resistance training | 1-3 sets, 8-12 reps, 50-85% 1RM | Upper + lower extremity |
| Supplemental O2 | During exercise if SpO2 <88% | Titrate to maintain SpO2 >90% |

**Respiratory Muscle Training (RMT):**

| Device Type | Protocol | Evidence |
|---|---|---|
| Threshold IMT | 30% of MIP, 2x daily, 15 min | Strong for dyspnea reduction |
| Normocapnic hyperpnea | 10-20 min, progressive | Moderate |
| PEP devices | During airway clearance | Strong for secretion clearance |

**Dyspnea Management Strategies:**
1. Pursed-lip breathing (prolongs exhalation, reduces air trapping)
2. Pacing strategies (activity/breathing coordination)
3. Positioning (tripod position, forward lean)
4. Fan/facial cooling (reduces sensation of breathlessness)
5. Energy conservation techniques

### 1.4 Pediatric Rehabilitation

Pediatric rehabilitation requires consideration of growth, development, family-centered care, and age-appropriate interventions.

#### 1.4.1 Developmental Disorders

**Common Conditions:**
- Developmental coordination disorder (DCD)
- Autism spectrum disorder (ASD) - motor impairments
- Down syndrome
- Spina bifida
- Duchenne muscular dystrophy
- Brachial plexus injury

**Intervention Principles:**
- Family-centered care model
- Play-based therapy approach
- Age-appropriate activities
- School-based vs. clinic-based services
- Individualized Education Program (IEP) coordination
- Neurodevelopmental treatment (NDT/Bobath) approach
- Task-specific training
- Bimanual training for upper limb

#### 1.4.2 Cerebral Palsy

**Classification Systems:**

| Classification | Basis | Levels |
|---|---|---|
| GMFCS | Gross motor function | I (walks without limitations) to V (transported) |
| MACS | Manual ability | I (handles objects easily) to III (handles with difficulty) |
| CFCS | Communication | I (effective sender/receiver) to V (seldom effective) |

**Evidence-Based Interventions:**

| Intervention | Target | Evidence Level | Ages |
|---|---|---|---|
| Constraint-Induced Movement Therapy (CIMT) | Upper limb | Strong | 18 months - adolescents |
| Bimanual Intensive Training (BIT) | Bilateral hand use | Strong | Similar to CIMT |
| Goal-Directed Training | Functional goals | Strong | All ages |
| Treadmill training | Gait | Moderate | Infants - adolescents |
| Botulinum toxin + casting | Spasticity/contracture | Strong | All ages |
| Selective dorsal rhizotomy | Spasticity reduction | Strong | GMFCS II-III, 4-10 years |
| Hip surveillance | Hip displacement prevention | Strong | All non-ambulatory |
| Strength training | Muscle weakness | Moderate | All ages |
| Body-weight supported training | Gait | Moderate | Variable |

**Early Intervention (0-2 years) - KEY WINDOW:**
- Highest neuroplasticity window
- Parent coaching models
- Baby-CIMT / Baby-BOET (bimanual occupational therapy)
- Goal-directed, activity-based
- Functional training embedded in daily routines
- Motor learning principles: repetition, variability, challenge

**Gross Motor Function Progression by GMFCS Level:**

```
GMFCS Level I: Independent ambulation, mild involvement
  - Focus: Speed, endurance, sport participation
  - Interventions: Strengthening, gait training, fitness

GMFCS Level II: Walks with limitations, uses rails on stairs
  - Focus: Gait quality, endurance, preventing decline
  - Interventions: Treadmill training, strength, balance

GMFCS Level III: Walks using hand-held mobility aids
  - Focus: Functional ambulation, transitions
  - Interventions: Walker/crutch training, power mobility intro

GMFCS Level IV: Self-mobility with limitations, may use powered mobility
  - Focus: Supported standing, transfers, powered mobility
  - Interventions: Standers, gait trainers, wheelchair skills

GMFCS Level V: Transported in manual wheelchair
  - Focus: Positioning, supported standing, caregiver training
  - Interventions: Custom seating, standers, passive ROM
```

### 1.5 Vestibular Rehabilitation

#### 1.5.1 Balance Disorders

**Vestibular System Components:**
- Peripheral: Semicircular canals (angular acceleration), otolith organs (linear acceleration/gravity)
- Central: Brainstem, cerebellum, cortical processing
- Integration: Visual, somatosensory, vestibular (multisensory integration)

**Common Vestibular Conditions:**

| Condition | Characteristics | Key Interventions |
|---|---|---|
| BPPV (benign paroxysmal positional vertigo) | Position-triggered vertigo, canalithiasis | Canalith repositioning maneuvers (Epley) |
| Unilateral vestibular hypofunction | Acute vestibular neuritis, labyrinthitis | Gaze stabilization, habituation, balance |
| Bilateral vestibular hypofunction | Aminoglycoside toxicity, Meniere's | Balance training, sensory substitution |
| Vestibular migraine | Migraine + episodic vertigo | Migraine management + vestibular rehab |
| PPPD (persistent postural-perceptual dizziness) | Chronic non-spinning dizziness | Habituation, CBT, gradual exposure |
| Concussion-related | Post-traumatic dizziness | Multimodal: vestibular, cervical, ocular |
| Age-related balance loss | Multifactorial | Multifactorial intervention |

#### 1.5.2 BPPV Management

**Canal Identification:**
- Posterior canal (90% of cases): Dix-Hallpike positive
- Horizontal canal: Supine roll test positive
- Anterior canal: Rare, Dix-Hallpike with torsional nystagmus

**Epley Maneuver (Posterior Canal):**
```
Step 1: Patient seated, turn head 45 degrees toward affected side
Step 2: Quickly lower to supine with head extended 20 degrees (off table)
Step 3: Hold 30-60 seconds after nystagmus stops
Step 4: Turn head 90 degrees to opposite side, hold 30-60 seconds
Step 5: Roll onto side, nose toward floor, hold 30-60 seconds
Step 6: Return to sitting, hold 30 seconds
```

#### 1.5.3 Vestibular Rehabilitation Therapy (VRT)

**Core Exercise Categories:**

1. **Gaze Stabilization Exercises:**
   - VOR x1: Fixate target while moving head
   - VOR x2: Target and head move opposite directions
   - Progression: Slow → fast, predictable → unpredictable, single → multiple targets
   - Goal: Clear vision during head movement

2. **Habituation Exercises:**
   - Exercises that provoke mild symptoms
   - Repeated to reduce symptom response
   - Visual motion stimulation
   - Position changes
   - Complex visual environments

3. **Balance Retraining:**
   - Static: Various surfaces, eyes open/closed
   - Dynamic: Weight shifting, reaching, turning
   - Gait: Various speeds, surfaces, head turns, dual tasks
   - Sensory manipulation: Foam, compliant surfaces

4. **Canalith Repositioning:**
   - Epley maneuver for posterior canal
   - BBQ roll for horizontal canal
   - Gufoni maneuver for horizontal canal
   - Brandt-Daroff exercises (home program)

**Progression Principles:**
- Start exercises at symptom level 3-5/10
- Progress by increasing: speed, complexity, dual-task demands, reducing sensory input
- Monitor symptom level: should not exceed 5/10 during exercise
- Symptoms should resolve within 15-20 minutes post-exercise

#### 1.5.4 Concussion Rehabilitation

**Multimodal Assessment Domains:**

| Domain | Assessment Tools | Interventions |
|---|---|---|
| Cervical | Joint mobility, muscle function | Manual therapy, strengthening |
| Vestibular | Dix-Hallpike, VOR testing | Gaze stabilization, repositioning |
| Ocular-motor | Convergence, saccades, smooth pursuit | Vision therapy exercises |
| Autonomic | HR response to exertion | Sub-symptom threshold training |
| Cognitive | ImPACT, SCAT5 | Graduated cognitive loading |
| Emotional | PHQ-9, GAD-7 | Counseling, graded activity |

**Return-to-Learn Protocol:**
```
Stage 1: Complete cognitive rest (24-48h)
Stage 2: Light mental activity (reading, limited screen time)
Stage 3: Part-time school with accommodations
Stage 4: Increased school hours, reduced workload
Stage 5: Return to full school with minimal accommodations
Stage 6: Full return to normal academics
```

### 1.6 Pelvic Floor Rehabilitation

#### 1.6.1 Conditions Addressed

| Condition | Population | Key Interventions |
|---|---|---|
| Stress urinary incontinence | Women postpartum, men post-prostatectomy | Pelvic floor muscle training (PFMT) |
| Urge urinary incontinence | Mixed populations | Bladder training, urge suppression |
| Mixed incontinence | Mixed | Combined PFMT + bladder training |
| Pelvic organ prolapse | Women | PFMT, lifestyle modification, pessary |
| Fecal incontinence | Mixed | Bowel management, anal sphincter exercises |
| Chronic pelvic pain | Mixed | Manual therapy, relaxation, biofeedback |
| Dyspareunia | Women | Desensitization, muscle retraining |
| Post-prostatectomy incontinence | Men | PFMT (pre + post surgery) |

#### 1.6.2 Pelvic Floor Muscle Training (PFMT)

**Evidence Base:**
- Strong evidence for stress urinary incontinence (Grade A recommendation)
- Moderate evidence for urge incontinence
- Preoperative PFMT reduces post-prostatectomy incontinence duration

**Assessment:**
- Digital vaginal/rectal examination
- Pelvic floor muscle palpation
- Observation (perineal descent/coat hanger sign)
- Biofeedback (surface EMG, manometry)
- PERFECT scheme (Power, Endurance, Repetitions, Fast contractions, Elevation, Coordination, Timing)

**Exercise Prescription:**

| Parameter | Protocol |
|---|---|
| Contraction type | Submaximal → maximal voluntary contraction |
| Hold time | 3-10 seconds (build progressively) |
| Repetitions | 8-12 per set |
| Sets | 3 sets per day |
| Fast contractions | 3-5 rapid contractions after each set |
| Position | Start supine → sitting → standing → functional |
| Duration | Minimum 3 months, ideally ongoing |

**Biofeedback Protocol:**
- Surface EMG electrodes (perianal/abdominal)
- Visual feedback on screen
- Goal: Isolate pelvic floor without abdominal/gluteal co-contraction
- Progress: Awareness → isolation → endurance → functional integration

#### 1.6.3 Post-Surgical Protocols

**Post-Prostatectomy Continence Recovery:**

```
Preoperative (2-6 weeks before surgery):
- Pelvic floor muscle education and training
- 3 sets of 8-10 contractions daily
- Biofeedback if available

Week 1-2 post-surgery:
- Resume PFMT when catheter removed
- Gentle contractions, pain-free
- Focus on quality over quantity

Week 2-6:
- Progressive loading of PFMT
- Functional activities (cough, lift with pre-contraction)
- Increase hold times and repetitions

Week 6-12:
- Maximal effort contractions
- Integration with daily activities
- Impact activities if continence improving

Month 3-6:
- Continue maintenance program
- Address any persistent issues
- Consider electrical stimulation if slow progress
```

## 2. Assessment Frameworks

This section details the most widely used, validated clinical assessment tools in rehabilitation practice. Each assessment includes administration guidelines, scoring, psychometric properties, and clinical interpretation.

### 2.1 Fugl-Meyer Assessment (FMA)

**Purpose:** Quantitative assessment of motor impairment, balance, sensation, and joint function in post-stroke hemiplegic patients. The most widely used measure of motor recovery after stroke.

**Domains Assessed:**

| Domain | Items | Score Range |
|---|---|---|
| Upper Extremity (FMA-UE) | 33 items | 0-66 |
| Lower Extremity (FMA-LE) | 17 items | 0-34 |
| Balance (sitting/standing) | 7 items | 0-14 |
| Sensation | 12 items | 0-24 |
| Joint Function (pain, ROM) | 2 items x multiple joints | 0-44 |
| **Total** | **155 items** | **0-226** |

**Scoring System:**
- 0 = Cannot perform
- 1 = Partially performs
- 2 = Performs fully

**FMA-UE Upper Extremity Subsections:**

| Section | Items | Points | Description |
|---|---|---|---|
| I. Reflex activity | 2 | 4 | Biceps, finger flexors |
| II. Volitional movement (flexor synergy) | 7 | 14 | Shoulder retraction, elevation, abduction, rotation; elbow flexion; forearm supination |
| III. Volitional movement (extensor synergy) | 5 | 10 | Shoulder adduction, internal rotation; elbow extension; forearm pronation |
| IV. Movement combining synergies | 6 | 12 | Hand to spine, shoulder flexion to 90, pronation-supination |
| V. Movement out of synergy | 6 | 12 | Shoulder abduction to 90, shoulder flexion to 90, supination |
| VI. Wrist function | 5 | 10 | Wrist stability, flexion/extension, circumduction |
| VII. Hand function | 7 | 14 | Finger mass flexion/extension, hook grasp, thumb adduction, cylindrical/spherical grasp |

**Motor Recovery Stages (Brunnstrom):**

| Stage | Clinical Description | Typical FMA-UE Score |
|---|---|---|
| I | Flaccid paralysis, no movement | 0 |
| II | Synergies appear, spasticity begins | 1-20 |
| III | Synergies performed voluntarily, spasticity peaks | 21-35 |
| IV | Movement outside synergy begins, spasticity decreases | 36-50 |
| V | More independent movement, coordination improves | 51-60 |
| VI | Near-normal coordination, minimal spasticity | 61-66 |
| VII | Normal function | 66 |

**Psychometric Properties:**

| Property | Value | Interpretation |
|---|---|---|
| Inter-rater reliability | ICC = 0.96-0.99 | Excellent |
| Intra-rater reliability | ICC = 0.98-0.99 | Excellent |
| Minimal Detectable Change (MDC) | UE: 5.3 points; LE: 5.0 points | Change > this value is real |
| Minimal Clinically Important Difference (MCID) | UE: 5.25 points (for CA group) | Meaningful improvement |
| SEM | UE: 1.9; LE: 1.8 | Standard error of measurement |

**Clinical Significance of FMA-UE Scores:**

| Score Range | Interpretation | Prognosis |
|---|---|---|
| <10 | Severe impairment | Limited recovery, compensatory focus |
| 10-30 | Moderate-severe | Some functional recovery possible |
| 30-45 | Moderate | Good potential for functional use |
| 45-55 | Moderate-mild | Dexterous hand function possible |
| >55 | Mild impairment | Near-normal function achievable |

**Administration Time:** 30-45 minutes for full assessment; 15-20 minutes for UE only

**Training Required:** Formal training recommended; certification available

### 2.2 Berg Balance Scale (BBS)

**Purpose:** Assess static and dynamic balance abilities in adults; predict fall risk; monitor change over time.

**Population:** Adults with balance impairment: stroke, Parkinson's, elderly, TBI, amputees

**14 Test Items:**

1. Sitting to standing
2. Standing unsupported
3. Sitting unsupported
4. Standing to sitting
5. Transfers
6. Standing with eyes closed
7. Standing with feet together
8. Reaching forward with outstretched arm
9. Retrieving object from floor
10. Turning to look behind
11. Turning 360 degrees
12. Placing alternate foot on step/stool
13. Standing with one foot in front
14. Standing on one foot

**Scoring:**
- 0 = Cannot perform
- 1 = Needs assist/considerable help
- 2 = Needs minimal help
- 3 = Performs independently
- 4 = Performs safely and independently

**Maximum Score: 56 points**

**Interpretation:**

| Score | Fall Risk Level | Functional Status |
|---|---|---|
| 0-20 | High fall risk | Wheelchair bound |
| 21-40 | Medium fall risk | Walking with assistance |
| 41-56 | Low fall risk | Independent ambulator |

**Specific Cutoffs:**

| Population | Cutoff | Sensitivity | Specificity |
|---|---|---|---|
| Elderly community-dwelling | <45 | 64% | 90% |
| Acute stroke | <40 | 77% | 86% |
| Parkinson's disease | <40 | 87% | 94% |
| General (2+ falls in 6 months) | <45 | - | - |

**Psychometric Properties:**

| Property | Value |
|---|---|
| Inter-rater reliability | ICC = 0.97-0.98 |
| Test-retest reliability | ICC = 0.97 |
| Internal consistency | Cronbach's alpha = 0.96 |
| MDC95 | 4.66 points (elderly), 6.12 (acute stroke) |
| MCID | 3.3 points (skilled nursing), 4-8 points (various populations) |
| SEM | 1.6 points |

**Administration Time:** 15-20 minutes
**Equipment:** Chair with armrests, stopwatch, ruler, step/stool

### 2.3 Timed Up and Go (TUG)

**Purpose:** Measure basic functional mobility; screen for fall risk; assess change over time.

**Procedure:**
1. Patient sits in standard armchair (seat height ~46 cm)
2. On "go," patient stands up, walks 3 meters, turns around, walks back, sits down
3. Time recorded from start command to seated completion

**Interpretation:**

| Time (seconds) | Interpretation |
|---|---|
| <10 | Normal, independent community ambulator |
| 10-14 | Normal for frail elderly, some limitations |
| 15-19 | Some impairment, at risk for falls |
| 20-29 | Significant impairment, fall risk |
| >=30 | Severe impairment, high fall risk, needs assistance |

**Dual-Task TUG (Advanced):**
- TUG + manual task (carrying cup of water)
- TUG + cognitive task (counting backwards by 3s, naming animals)
- Dual-task cost = (Dual-task time - Single-task time) / Single-task time x 100%
- Dual-task cost > 20% indicates increased fall risk

**Psychometric Properties:**

| Property | Value |
|---|---|
| Test-retest reliability | ICC = 0.95 |
| Inter-rater reliability | ICC = 0.99 |
| MDC95 | 3.5 seconds |
| MCID | 3.3 seconds (community-dwelling elderly), 1.8 (neurological) |

**Administration Time:** <5 minutes

---

### 2.4 Six-Minute Walk Test (6MWT)

**Purpose:** Measure functional exercise capacity; assess submaximal level of functional capacity; monitor response to interventions.

**Procedure:**
1. 30-meter straight course marked (or shorter with more turns)
2. Patient walks at self-paced speed for 6 minutes
3. Encouragement provided at standardized intervals (every minute)
4. Total distance recorded; can also record HR, SpO2, RPE
5. Stop test if patient requests or shows signs of distress

**Standardized Encouragement Script:**
- "You are doing well. You have [X] minutes to go."
- Do NOT coach to walk faster

**Equipment:**
- 30-meter hallway or gym
- Cones/markers
- Stopwatch
- Pulse oximeter (optional)
- Borg RPE scale
- Blood pressure monitor
- Chair at each end for rest if needed

**Reference Values:**

| Population | Mean Distance | Significant Change |
|---|---|---|
| Healthy men (60-70 years) | 500-600m | >54m |
| Healthy women (60-70 years) | 450-550m | >54m |
| Chronic heart failure | 300-450m | >45m |
| COPD | 350-500m | >35m |
| Post-stroke | 200-350m | >34m |
| Parkinson's disease | 300-450m | >35m |

**MCID by Condition:**

| Condition | MCID |
|---|---|
| COPD | 25-35 meters |
| Chronic heart failure | 45-50 meters |
| Stroke | 34-40 meters |
| Healthy elderly | 50-80 meters |

**Predicted Distance Formulas:**

For healthy adults:
- Men: 6MWD = (7.57 x height_cm) - (5.02 x age) - (1.76 x weight_kg) - 309 m
- Women: 6MWD = (2.11 x height_cm) - (2.29 x weight_kg) - (5.78 x age) + 667 m

**Psychometric Properties:**

| Property | Value |
|---|---|
| Test-retest reliability | ICC = 0.94-0.98 |
| MDC95 | ~50-80 meters (population-dependent) |
| MCID | 25-54 meters (condition-dependent) |

**Administration Time:** 10-15 minutes (including setup, rest, test)

---

### 2.5 10-Meter Walk Test (10MWT)

**Purpose:** Assess gait speed over a short distance; strong predictor of functional outcomes, mortality, and healthcare utilization.

**Procedure:**
1. 14-meter straight course (2m acceleration + 10m timed + 2m deceleration)
2. Markers at 2m, 12m (10m timed zone)
3. Patient walks at comfortable pace, time the middle 10m
4. Repeat at fast pace ("walk as fast as you safely can")
5. Record time for 10m, calculate speed (m/s)

**Self-Selected vs. Maximum Gait Speed:**
- Self-selected: Reflects habitual walking speed
- Maximum: Reflects reserve capacity and safety margin

**Interpretation:**

| Speed (m/s) | Functional Level | Community Ambulation |
|---|---|---|
| 0-0.2 | Household ambulator | No |
| 0.2-0.4 | Limited community | Limited |
| 0.4-0.6 | Community ambulator | Partial |
| 0.6-0.8 | Full community | Yes |
| 0.8-1.0 | Normal range (elderly) | Yes |
| >1.0 | Normal range (adults) | Yes |

**Clinical Significance:**

| Speed Threshold | Clinical Meaning |
|---|---|
| <0.8 m/s | Increased fall risk, cognitive decline risk |
| <1.0 m/s | Limited community ambulation |
| >1.2 m/s | Cross streets safely, independent community |
| >1.4 m/s | Normal young adult speed |

**Psychometric Properties:**

| Property | Value |
|---|---|
| Test-retest reliability | ICC = 0.91-0.99 |
| Inter-rater reliability | ICC = 0.95-0.99 |
| MDC95 | 0.1-0.2 m/s |
| MCID | 0.05-0.10 m/s (small), 0.10-0.15 m/s (substantial) |

**Administration Time:** 5 minutes

---

### 2.6 Modified Ashworth Scale (MAS)

**Purpose:** Measure spasticity (velocity-dependent increase in muscle tone) in patients with upper motor neuron lesions.

**Grading System:**

| Grade | Description |
|---|---|
| 0 | No increase in muscle tone |
| 1 | Slight increase, catch and release, or minimal resistance at end ROM |
| 1+ | Slight increase, catch followed by minimal resistance through remainder (<50%) of ROM |
| 2 | More marked increase through most of ROM, but affected part easily moved |
| 3 | Considerable increase, passive movement difficult |
| 4 | Affected part rigid in flexion or extension |

**Administration:**
1. Patient in relaxed, comfortable position
2. Passively move limb at specified velocity ("as fast as possible")
3. Feel for catch, resistance, or clasp-knife phenomenon
4. Test at multiple joints as indicated

**Important Notes:**
- MAS measures resistance to passive movement (not true spasticity)
- Confounded by contracture, weakness, co-contraction
- More reliable at elbow flexors than lower extremity
- Tardieu Scale may be more valid measure of spasticity (measures at different velocities)

**Psychometric Properties:**

| Property | Value |
|---|---|
| Inter-rater reliability | Kappa = 0.46-0.85 (moderate to excellent) |
| Test-retest reliability | Kappa = 0.63-0.91 |
| Correlation with neurophysiological measures | Moderate |

**Administration Time:** 10-15 minutes for full assessment

### 2.8 Functional Independence Measure (FIM)

**Purpose:** Assess functional disability and burden of care in performing activities of daily living (ADLs). Used for outcome measurement, reimbursement determination, and care planning.

**18 Items across 6 domains:**

| Domain | Items | Points |
|---|---|---|
| Self-care | Eating, grooming, bathing, upper body dressing, lower body dressing, toileting | 6 x 7 = 42 |
| Sphincter control | Bladder management, bowel management | 2 x 7 = 14 |
| Mobility | Bed/chair/wheelchair transfer, toilet transfer, tub/shower transfer | 3 x 7 = 21 |
| Locomotion | Walk/wheelchair, stairs | 2 x 7 = 14 |
| Communication | Comprehension, expression | 2 x 7 = 14 |
| Social cognition | Social interaction, problem solving, memory | 3 x 7 = 21 |
| **Total** | **18 items** | **Maximum: 126** |

**Scoring per Item:**

| Score | Level | Description |
|---|---|---|
| 7 | Complete independence | Timely, safe, no assistive device |
| 6 | Modified independence | Device required, extra time |
| 5 | Supervision | No physical help, standby/cueing |
| 4 | Minimal assist | Patient does 75%+ of task |
| 3 | Moderate assist | Patient does 50-74% of task |
| 2 | Maximal assist | Patient does 25-49% of task |
| 1 | Total assist | Patient does <25% or unable |

**Motor vs. Cognitive Subscales:**
- Motor FIM: Self-care (6) + Sphincter (2) + Mobility (3) + Locomotion (2) = 13 items, max 91
- Cognitive FIM: Communication (2) + Social cognition (3) = 5 items, max 35

**Interpretation:**

| Total Score | Dependence Level |
|---|---|
| 108-126 | Modified independence or better |
| 90-107 | Mild dependence |
| 72-89 | Moderate dependence |
| 54-71 | Severe dependence |
| 18-53 | Very severe dependence |

**FIM Efficiency:**
- FIM gain / Length of stay
- Measures rehabilitation efficiency (points gained per day)
- Typical values: 1-3 FIM points/day in acute rehabilitation

**Psychometric Properties:**

| Property | Value |
|---|---|
| Inter-rater reliability | ICC = 0.86-0.95 |
| Test-retest reliability | ICC = 0.95 |
| Internal consistency | Cronbach's alpha = 0.93-0.98 |
| Floor effect (stroke) | Yes (ceiling at admission for some items) |
| Ceiling effect (community) | Yes |

**Administration Time:** 20-30 minutes (by trained assessor)

### 2.9 Dynamometry (Strength Testing)

**Purpose:** Objectively measure muscle strength using handheld or fixed dynamometry.

#### 2.9.1 Handheld Dynamometry (HHD)

**Common Muscle Groups Tested:**

| Joint | Movement | Patient Position | Stabilization |
|---|---|---|---|
| Shoulder | Abduction, flexion, ER, IR | Supine or sitting | Torso |
| Elbow | Flexion, extension | Sitting | Upper arm |
| Wrist | Flexion, extension, grip | Sitting | Forearm |
| Hip | Flexion, extension, abduction, adduction | Supine/side-lying | Pelvis |
| Knee | Flexion, extension | Sitting/supine | Thigh |
| Ankle | Dorsiflexion, plantarflexion, inversion | Sitting | Lower leg |

**Standardized Testing Protocol:**
1. Explain test procedure
2. Position patient in standardized position
3. Stabilize proximal body part
4. Apply force perpendicular to limb segment
5. Patient performs maximal voluntary isometric contraction (MVIC)
6. Hold 3-5 seconds, record peak force
7. 3 trials with 30-second rest, record best
8. Compare to normative values (age, gender matched)

**Grip Strength (Jamar Dynamometer):**

| Position | Procedure |
|---|---|
| 5 handle positions tested | Position 2 typically yields maximum |
| 3 squeezes per hand | Record best of 6 trials |
| Elbow at 90 degrees | Shoulder adducted, neutral rotation |
| Seated position | Forearm and wrist in neutral |

**Grip Strength Norms ( kg):**

| Age | Male (dominant) | Female (dominant) |
|---|---|---|
| 20-29 | 46-56 | 27-33 |
| 30-39 | 48-58 | 29-35 |
| 40-49 | 46-56 | 27-33 |
| 50-59 | 42-52 | 24-30 |
| 60-69 | 37-45 | 21-27 |
| 70-79 | 31-39 | 18-24 |

**Interpretation:**
- <5th percentile: Significantly impaired
- Grip strength <30 kg (male) or <20 kg (female): Functional limitation indicator
- Asymmetry >10-15% between sides: Clinically meaningful

#### 2.9.2 Isokinetic Dynamometry

**Purpose:** Gold standard for measuring muscle strength at controlled angular velocities.

| Parameter | Description |
|---|---|

| Average power | Work done per unit time |
| Total work | Work across all repetitions |
| Endurance ratio | Fatigue index (work first 3 reps / last 3 reps) |

**Common Testing Protocols:**

| Joint | Speeds (degrees/s) | Repetitions |
|---|---|---|
| Knee | 60, 180, 300 | 5 at each speed |

| Ankle | 30, 60, 120 | 5 at each speed |

| Property | Value |
|---|---|
| Inter-rater reliability | ICC = 0.76-0.96 |
| Test-retest reliability | ICC = 0.81-0.97 |

## 3. Exercise Prescription

### 3.1 The FITT-VP Principle

The FITT-VP framework provides a systematic approach to exercise prescription, adapted from the American College of Sports Medicine (ACSM) guidelines.

| Component | Definition | Clinical Application |
|---|---|---|
| **F** - Frequency | How often exercise is performed | Sessions per week, rest days between |
| **I** - Intensity | How hard the exercise is | % HRmax, % 1RM, RPE, METs, %VO2 reserve |
| **T** - Time | Duration of each session | Minutes per session, includes warm-up/cool-down |
| **T** - Type | Mode of exercise | Aerobic, resistance, flexibility, neuromuscular |
| **V** - Volume | Total amount of exercise performed | Reps x sets x load (resistance); duration x intensity (aerobic) |
| **P** - Progression | Rate of advancement | Based on adaptation, tolerance, goals |

#### 3.1.1 Frequency Guidelines by Modality

| Exercise Type | Beginner | Intermediate | Advanced |
|---|---|---|---|
| Aerobic | 3 days/week | 4-5 days/week | 5-7 days/week |
| Resistance | 2 days/week | 3 days/week | 4-5 days/week |
| Flexibility | 2-3 days/week | 3-4 days/week | Daily |
| Balance/Neuromuscular | 2-3 days/week | 3-4 days/week | 4-5 days/week |

#### 3.1.2 Intensity Prescription Methods

**Cardiovascular Exercise Intensity Zones:**

| Zone | % HRmax | % HRR | % VO2max | RPE (6-20) | Description |
|---|---|---|---|---|---|
| Very light | <50% | <30% | <37% | <10 | Recovery, warm-up |
| Light | 50-63% | 30-39% | 37-45% | 10-11 | Comfortable, can converse |
| Moderate | 64-76% | 40-59% | 46-63% | 12-13 | Somewhat hard, steady pace |
| Vigorous | 77-95% | 60-89% | 64-90% | 14-16 | Hard, conversation difficult |
| Near-maximal | >96% | >90% | >90% | 17-19 | Very hard, sustainable briefly |

**Resistance Training Intensity:**

| Goal | Intensity (% 1RM) | Repetitions | Sets | Rest Between Sets |
|---|---|---|---|---|
| Strength (beginner) | 60-70% | 8-12 | 1-3 | 2-3 minutes |
| Strength (advanced) | 80-95% | 1-6 | 3-5 | 2-5 minutes |
| Power (velocity-focused) | 30-60% | 3-6 | 3-5 | 3-5 minutes |
| Hypertrophy | 70-85% | 6-12 | 3-6 | 60-90 seconds |
| Muscular endurance | 40-60% | 15-25 | 2-3 | 30-60 seconds |

**Target Heart Rate Calculation:**
```
Standard method: HRmax = 220 - age
Tanaka method:   HRmax = 208 - (0.7 x age)
Nes method:      HRmax = 211 - (0.64 x age)

Heart Rate Reserve (HRR) = HRmax - HRrest
Target HR = (Intensity% x HRR) + HRrest

Karvonen formula example for 50-year-old, HRrest=60, target 60% HRR:
HRmax = 220 - 50 = 170
HRR = 170 - 60 = 110
Target HR = (0.60 x 110) + 60 = 66 + 60 = 126 bpm
```

#### 3.1.4 Progression Models

**Conservative Progression (Deconditioned, Elderly, Chronic Disease):**
- Increase duration by 10% per week
- Increase intensity only after 2-3 weeks at current level
- Frequency increased last

**Moderate Progression (Generally Healthy):**
- Increase duration by 10-20% per week
- Intensity increased every 2-3 weeks
- Volume follows 10% rule (no more than 10% weekly increase)

**Aggressive Progression (Athletes, Young Adults):**
- Duration and intensity can increase weekly
- Periodization models employed
- Supervised progression with performance monitoring

**Clinical Decision Rules for Progression:**

| Factor | Progress if | Hold/Regress if |
|---|---|---|
| Pain | No increase during/after | Pain increases >2/10 |
| Fatigue | Recovers between sessions | Persistent fatigue 24h+ |
| Performance | Improving or stable | Declining |
| Symptoms | Stable/improving | New or worsening symptoms |
| Adherence | >80% sessions | <60% sessions |

### 3.2 Neuroplasticity-Based Exercises

Neuroplasticity-based exercise design leverages the brain's capacity to reorganize neural pathways in response to experience, learning, and injury.

#### 3.2.1 Repetitive Task Practice (RTP)

**Core Principles:**
- High repetition: 300-600 repetitions per session
- Task-specific training: Practice actual functional tasks
- Progressive challenge: Increase difficulty as skill improves
- Active engagement: Patient-initiated, attention-demanding
- Feedback: Knowledge of results and performance

**Dose-Response Evidence for Upper Extremity:**

| Repetition Threshold | Evidence Source | Outcome |
|---|---|---|
| 300 reps/session | Wolf et al. (EXCITE trial) | Significant motor improvement |
| 600 reps/session | Nudo et al. (animal models) | Maximum cortical reorganization |
| 15,000 reps/month | Ward et al. | Required for meaningful change |
| ~32,000 reps/6-week program | Ballester et al. | Clinically significant improvement |

**RTP Implementation Strategies:**

1. **Block practice:** Repeated practice of single task (good for early learning)
2. **Variable practice:** Multiple tasks in varied order (better retention/transfer)
3. **Random practice:** Unpredictable task sequence (enhances motor learning)
4. **Increasing challenge:** Progress difficulty within same task category

**Sample RTP Tasks by Difficulty Level:**

| Simple | Intermediate | Advanced |
|---|---|---|
| Reaching for cone | Stacking cups | Pouring water |
| Tabletop sliding | Button/unbutton | Using key in lock |
| Grasp and release | Turning pages | Writing with pen |
| Wrist weight shifts | Picking up coins | Tying shoelace |
| Hand to table | Hand to mouth | Fastening belt |

#### 3.2.2 Constraint-Induced Movement Therapy (CIMT)

**Original Protocol (CIMT - Classic):**
- Restraint of unaffected limb: 90% of waking hours for 2 weeks
- Intensive training: 6 hours/day of structured task practice
- Shaping methodology: Tasks progressively increased in difficulty
- Transfer package: Problem-solving to integrate into daily life

**Modified CIMT (mCIMT) - More Clinically Accessible:**
- Restraint: 5-6 hours/day, 5 days/week for 2-3 weeks
- Training: 30 minutes to 2 hours/day (per session)
- Home practice component required

**Inclusion Criteria:**
- 20+ degrees active wrist extension
- 10+ degrees active finger extension at 2 digits
- Some ability to move affected limb
- Sufficient cognition to follow instructions
- Sufficient balance (if training involves standing)

**Transfer Package Components:**
1. Motor Activity Log (MAL) - self-report of real-world use
2. Problem-solving for obstacles
3. Home practice assignment
4. Patient-therapist contract for adherence
5. Daily diary of affected limb use

#### 3.2.3 Motor Imagery & Mental Practice

**Types of Motor Imagery:**
- **Internal (kinesthetic):** Feeling the movement from first-person perspective
- **External (visual):** Watching oneself perform from third-person perspective

**Evidence-Based Protocol:**
- 15-20 minutes per session
- Combined with physical practice (superior to either alone)
- First-person (internal) imagery preferred for motor learning
- Audio-guided scripts can enhance adherence
- Timing: Before, during, or after physical practice

**Contraindications:**
- Significant cognitive impairment
- Severe depression or anxiety that worsens with imagery
- Inability to generate motor images (assessed with VMIQ)

### 3.3 Balance Training

#### 3.3.1 Balance Classification

| Balance Type | Description | Examples |
|---|---|---|
| Static | Maintaining position without movement | Standing still, single-leg stance |
| Dynamic | Maintaining balance during movement | Walking, reaching, turning |
| Reactive | Responding to external perturbations | Platform perturbations, nudges |
| Anticipatory | Preparing for self-initiated movement | Stepping up, lifting object |
| Proactive | Planning movements to avoid instability | Avoiding obstacle while walking |

#### 3.3.2 Balance Training Progression Framework

**Progression Parameters:**
1. Base of support (BOS): Wide → narrow → tandem → single-leg
2. Center of gravity (COG): Low → high
3. Sensory input: All senses → reduced (eyes closed, foam surface)
4. Cognitive load: Single task → dual task
5. Surface: Firm → compliant → unstable
6. Perturbations: None → predictable → unpredictable

**Static Balance Exercise Progression:**

```
Level 1:  Wide stance, firm surface, eyes open, arms free
Level 2:  Narrow stance, firm surface, eyes open
Level 3:  Tandem stance, firm surface, eyes open
Level 4:  Single-leg stance, firm surface, eyes open (10s goal)
Level 5:  Wide stance, eyes closed
Level 6:  Tandem stance, eyes closed
Level 7:  Single-leg stance, eyes closed (5s goal)
Level 8:  Wide stance on foam/uneven surface
Level 9:  Tandem on foam
Level 10: Single-leg on foam
Level 11: Single-leg on foam, eyes closed
Level 12: Single-leg on foam, head turns, cognitive dual-task
```

**Dynamic Balance Exercise Progression:**

```
Level 1:  Weight shifts (side-to-side, forward-backward)
Level 2:  Reaching outside BOS (forward, lateral, multi-directional)
Level 3:  Mini-squats with weight shift
Level 4:  Step-taps in multiple directions
Level 5:  Step-ups (2-4 inches)
Level 6:  Lunges (forward, lateral)
Level 7:  Tandem walking
Level 8:  Single-leg stance with functional reach
Level 9:  Step-overs (obstacles)
Level 10: Lateral stepping, crossover steps
Level 11: Sit-to-stand from varied heights
Level 12: Reactive: perturbations during standing
Level 13: Reactive: perturbations during gait
Level 14: Complex: sport-specific balance challenges
```

#### 3.3.3 Dual-Task Balance Training

**Rationale:** Real-world balance requires simultaneous cognitive/motor processing. Dual-task training improves divided attention capabilities.

| Motor Task | Cognitive Task | Motor + Motor Task |
|---|---|---|
| Standing balance | Counting backwards by 3s | Stand + catch/throw ball |
| Tandem walking | Naming animals | Walk + carry tray with cup |
| Single-leg stance | Stroop color-word test | Walk + kick ball |
| Heel-toe walking | Serial 7s subtraction | Walk + head turns |
| Obstacle walking | Recall memory test | Stand + reach for objects |

**Measuring Dual-Task Cost:**
```
Cognitive cost = (Dual-task cognitive score - Single-task cognitive score) / Single-task score x 100%
Motor cost = (Dual-task motor performance - Single-task motor performance) / Single-task performance x 100%

Dual-task cost > 15-20% suggests impaired dual-task processing
```

### 3.4 Gait Training

#### 3.4.1 Gait Training Progression

**Pre-Gait Readiness Criteria:**
- Standing tolerance: 30+ minutes
- Weight-bearing tolerance on affected limb
- Sufficient lower extremity strength (3+/5 in key muscles)
- Dynamic standing balance

**Overground Gait Training Progression:**

| Stage | Assistive Device | Distance | Focus |
|---|---|---|---|
| 1 | Parallel bars | 10-20 feet | Weight acceptance, symmetry |
| 2 | Walker | 50-100 feet | Gait pattern, safety |
| 3 | Two crutches | 100-200 feet | Arm-leg coordination |
| 4 | Single crutch/cane | 200+ feet | Normalization |
| 5 | Unassisted (indoors) | Unlimited | Quality, speed |
| 6 | Unassisted (community) | Unlimited | Endurance, varied surfaces |

#### 3.4.2 Body-Weight Supported Treadmill Training (BWSTT)

**Parameters:**

| Parameter | Initial Setting | Progression |
|---|---|---|
| Body weight support | 30-40% of body weight | Reduce by 5-10% per session as tolerated |
| Treadmill speed | 0.5-1.0 mph | Increase by 0.1-0.2 mph per session |
| Duration | 10-15 minutes | Progress to 20-30 minutes |
| Handrail use | Both hands | Progress to one hand, then no hands |
| Therapist assistance | 2-3 therapists (physical guidance) | Reduce assistance as gait improves |

**Evidence Summary:**
- Stroke: Moderate evidence for ambulatory patients (LEAPS trial)
- SCI: Strong evidence for incomplete injuries (activity-based therapy)
- Parkinson's: Moderate evidence for gait quality
- Pediatric CP: Moderate evidence for GMFCS III-IV

#### 3.4.3 Gait Quality Parameters

| Parameter | Normal Value | Target for Rehab |
|---|---|---|
| Cadence | 100-120 steps/min | Functional cadence >80 steps/min |
| Step length | 70-80 cm (varies by height) | Symmetry index 90-110% |
| Stride length | 140-160 cm | Symmetry index 90-110% |
| Base of support | 5-10 cm | <15 cm |
| Double support time | 10-15% of gait cycle | Reduce toward normal |
| Gait speed | 1.2-1.5 m/s | >0.8 m/s for community ambulation |
| Step time symmetry | 90-110% | Within 10% between limbs |

### 3.5 Strengthening

#### 3.5.1 Progressive Resistance Training (PRT)

**DeLorme's Progressive Resistance Exercise (PRE) Protocol:**

| Set | Load (% of 10RM) | Repetitions | Purpose |
|---|---|---|---|
| 1 | 50% | 10 | Warm-up |
| 2 | 75% | 10 | Preparation |
| 3 | 100% | 10 (maximal) | Overload |

**Oxford (DAPRE) Technique:**

| Set | Load (% of working weight) | Repetitions |
|---|---|---|
| 1 | 50% | 10 |
| 2 | 75% | 6 |
| 3 | 100% | Maximum possible |
| 4 | Adjusted based on Set 3 performance | Maximum possible |

**Set 4 Load Adjustment:**

| Set 3 Reps Completed | Set 4 Weight Adjustment |
|---|---|
| 0-2 | Reduce 5-10 lbs |
| 3-4 | Reduce 0-5 lbs |
| 5-6 | Same weight |
| 7-10 | Increase 5-10 lbs |
| 11+ | Increase 10-15 lbs |

#### 3.5.2 Eccentric Training

**Rationale:** Eccentric loading produces greater force with less metabolic demand; important for tendinopathy treatment.

**Heavy Slow Resistance (HSR) Protocol for Tendinopathy:**
- 3 sets x 15 reps at 70% 1RM
- Progress to 3 sets x 12 at 75%
- Progress to 3 sets x 10 at 80%
- 3 seconds concentric, 3 seconds eccentric
- 3x/week for 12+ weeks
- Pain during exercise acceptable if settles within 24 hours

**Nordic Hamstring Curl Protocol (for hamstring injury prevention):**

| Week | Sets x Reps | Frequency | Notes |
|---|---|---|---|
| 1 | 2 x 5 | 1x/week | Partner-assisted, limited range |
| 2-3 | 2 x 6 | 2x/week | Progress range |
| 4-5 | 3 x 6-8 | 2x/week | Full range with assist |
| 6-8 | 3 x 8-10 | 2x/week | Minimal assist |
| 9-10 | 3 x 8-10 | 1x/week | Unassisted (maintenance) |

#### 3.5.3 Isometric Training

**Applications:**
- Pain modulation (isometrics reduce pain for 45+ minutes post-exercise)
- Early tendon loading (especially for reactive tendinopathy)
- Post-surgical immobilization (maintain neural drive)
- Blood pressure management (lower cardiovascular response)

**Protocol for Tendinopathy Pain Modulation:**
- 5 x 45-second isometric holds
- 70-80% MVIC
- 2-minute rest between holds
- Perform once daily or before loading exercise

### 3.7 Flexibility Training

#### 3.7.1 Static Stretching

| Parameter | Recommendation |
|---|---|
| Duration | 15-30 seconds per stretch |
| Repetitions | 2-4 per muscle group |
| Frequency | 2-3 days minimum, ideally daily |
| Intensity | Mild tension, NOT pain |
| Position | Stable, comfortable, targeted |

**Physiological Effects:**
- Increases passive range of motion (tolerance-based)
- May reduce muscle-tendon stiffness
- No significant evidence for injury prevention
- Best for: Post-exercise cool-down, rehabilitation, relaxation

#### 3.7.2 Dynamic Stretching

**Characteristics:**
- Controlled movement through full ROM
- Sport-specific movement patterns
- Gradual increase in amplitude and speed
- Prepares neuromuscular system for activity

**Examples by Activity:**

| Activity | Dynamic Stretches |
|---|---|
| Running | Leg swings, walking lunges, high knees, butt kicks |
| Swimming | Arm circles, shoulder rolls, trunk rotations |
| Cycling | Leg swings, hip circles, walking quadricep stretch |
| Tennis | Torso rotations, arm circles, side lunges |

#### 3.7.3 Proprioceptive Neuromuscular Facilitation (PNF)

**Common Techniques:**

| Technique | Procedure | Best For |
|---|---|---|
| Hold-Relax (HR) | Stretch → isometric contraction → relax → stretch further | General flexibility |
| Contract-Relax (CR) | Same as HR but with concentric contraction | Overcoming tightness |
| Hold-Relax with Agonist Contraction (HRAC) | HR + active contraction of antagonist | Greater ROM gains |
| Contract-Relax-Antagonist-Contract (CRAC) | Combined CR + agonist contraction | Maximum ROM improvement |

**PNF Protocol (Hold-Relax example):**
1. Position limb at comfortable stretch point
2. Patient isometrically contracts tight muscle for 6-10 seconds
3. Patient relaxes completely
4. Stretch is increased to new barrier
5. Repeat 3-4 times

### 3.8 Aquatic Therapy

#### 3.8.1 Physical Properties of Water

| Property | Clinical Application | Effect |

### 4.1 Protocol 1: Post-Stroke Upper Extremity (6-Week Protocol)

**Target Population:** Adults with hemiparetic upper extremity following stroke; FMA-UE score 10-50; medically stable

#### Week 1-2: Foundation Phase

**Goals:**
1. Initiate movement in affected upper extremity
2. Reduce learned non-use patterns
3. Establish motor control at proximal joints
4. Begin bilateral training

**Exercises (5 days/week, 60-90 minutes/session):**

| Exercise | Sets x Reps | Parameters | Notes |
|---|---|---|---|
| Table-top sliding (forward, lateral) | 3 x 10 each direction | Upright sitting, weight-bearing through UE | Promote scapular protraction |
| Shoulder flexion (AAROM) | 3 x 10 | Patient-assisted with unaffected hand | Within pain-free ROM |
| Shoulder abduction (AAROM) | 3 x 10 | Patient-assisted or pulley | <90 degrees initially |
| Elbow flexion/extension (AAROM) | 3 x 10 | Gravity-assisted or supported | Promote active control |
| Wrist circumduction | 3 x 10 clockwise/counter | Supported forearm on table | Active-assisted |
| Finger flexion/extension | 5 x 10 | Active-assisted, emphasize extension | Prevent flexion contracture |
| Weight-bearing through affected UE | 3 x 30-60s hold | Leaning on table at various angles | Increase tolerance |
| Bilateral symmetrical reaching | 3 x 10 | Both hands to target simultaneously | Mirror therapy setup |
| Mirror therapy | 15-20 min session | Watch unaffected hand in mirror | Appear as affected hand moving |
| NMES to wrist/finger extensors | 30 min | 35-50 Hz, on:off 5:15, sensory-motor threshold | If available |

**Daily Home Program (30 minutes):**
- Active-assisted ROM exercises (10 minutes)
- Mirror therapy (10 minutes)
- Weight-bearing activities (10 minutes)

**Progression Criteria (to Week 3):**
- Some active movement in proximal joints (shoulder, elbow)
- Tolerates table-top weight-bearing 60 seconds
- Pain <3/10 during exercises
- No adverse events

---

#### Week 3-4: Activation Phase

**Goals:**
1. Increase active range of motion
2. Introduce task-specific practice
3. Begin strengthening of innervated muscles
4. Increase repetitions toward neuroplasticity threshold

**Exercises (5 days/week, 90 minutes/session):**

| Exercise | Sets x Reps | Parameters | Progression |
|---|---|---|---|
| Reaching exercises (multi-directional) | 5 x 15 | Unassisted active reaching to targets | Increase distance, vary target height |
| Pushing/pulling activities | 4 x 15 | Push light objects across table | Increase weight of objects |
| Grasp and release (various objects) | 5 x 10 each object | Different sizes, shapes, weights | Progress from foam blocks to coins |
| Weight-bearing with weight shifts | 4 x 10 | Forward/sideward weight shifts on table | Increase amplitude |
| CIMT preparation (if candidate) | Per protocol | Practice with mitt on unaffected hand 2h/day | Functional task practice |
| Bilateral arm training | 4 x 15 | Both arms performing same movement | Progress to asymmetrical |
| Wrist curls with light weight | 3 x 15 | 0.5-1 lb, supported forearm | Increase weight gradually |
| Finger opposition exercises | 3 x 10 each finger | Thumb to each fingertip | Increase speed |
| Fine motor tasks (pegboard) | 10 min continuous | Pick up and place pegs | Time trials for motivation |
| Functional task practice | 20 min | Comb hair, drink from cup, button | Real objects, daily activities |

**Target:** 300+ repetitions per session of task practice

**Progression Criteria (to Week 5):**
- Active ROM improving in all joints
- Can lift 1-2 lb with affected hand
- Demonstrates grasp and release of varied objects
- FMA-UE improvement of at least 5 points

---

#### Week 5-6: Integration Phase

**Goals:**
1. Maximize functional use in daily activities
2. Achieve 600+ repetitions per session
3. Transition to independent home program
4. Bimanual task performance

**Exercises (5 days/week, 90-120 minutes/session):**

| Exercise | Sets x Reps | Parameters | Progression |
|---|---|---|---|
| Repetitive task practice (RTP) | 30 min blocks | 600+ reps/session, varied tasks | Progressive difficulty |
| Advanced grasp patterns | 5 x 10 each | Spherical, cylindrical, tip pinch | Functional objects |
| In-hand manipulation | 3 x 10 | Translate objects within hand | Coins, pencils, small balls |
| Bilateral functional tasks | 20 min | Open jars, fold towels, food prep | Daily living activities |
| Upper extremity ergometry | 10 min | Low resistance, 60-80 RPM | Monitor for fatigue |
| Constraint-induced practice | Per CIMT protocol | 2-3 hours affected UE use with mitt | Home-based CIMT |
| Overhead reaching | 4 x 15 | Place objects on shelf | Increase height |
| Fast grasp and release | 5 x 15 | Speed-focused, timed tasks | Reduce time to complete |
| Handwriting practice | 10 min | Write name, sentences | Progress to paragraphs |
| Transfer training | 10 min | Carry objects while walking | Safety emphasis |

**Home Program Transition:**
- Motor Activity Log (MAL) daily completion
- 1-hour structured practice daily
- Functional use goals for each day

**Outcome Measures:**

| Measure | Timing | Target |
|---|---|---|
| FMA-UE | Baseline, 3 weeks, 6 weeks | MCID >5.25 points |

### 4.7 Protocol 7: Cardiac Rehabilitation (12-Week Protocol)

**Target Population:** Post-cardiac event (MI, CABG, PCI, stable angina, valve surgery, heart failure); Phase II cardiac rehabilitation

**Prerequisites:**
- Physician clearance
- Cardiac risk stratification completed
- Baseline exercise testing (symptom-limited or submaximal)
- Stable condition for minimum 2 weeks (post-event)

#### Week 1-3: Initiation & Monitoring

**Goals:**
1. Establish exercise tolerance baseline
2. Initiate low-intensity aerobic training
3. Begin resistance training education
4. Risk factor education
5. Medication optimization

**Initial Exercise Testing (Symptom-Limited):**

| Test | Purpose | Endpoints |
|---|---|---|
| Modified Bruce Protocol | Assess functional capacity | Symptoms, ECG changes, BP response |
| 6-Minute Walk Test | Functional capacity | Distance, HR, SpO2, symptoms |
| Metabolic equivalents (METs) | Activity prescription | Baseline functional capacity |
| HR/BP response to exercise | Autonomic response | Abnormal responses identified |
| Rate-pressure product | Myocardial oxygen demand | RPP = HR x SBP |

**Risk Stratification (AACVPR):**

| Factor | High Risk | Low Risk |
|---|---|---|
| LVEF | <40% | >50% |
| Complex ventricular arrhythmias | Present | Absent |
| Angina | Unstable or provoked easily | None or stable |
| Functional capacity | <5 METs | >7 METs |
| BP response to exercise | Abnormal | Normal |
| Clinical depression | Present | Absent |
| Diabetes | Uncontrolled | Controlled/absent |
| Kidney disease | Stage 3b+ | Normal |
| Previous cardiac arrest | Yes | No |
| Multi-vessel disease | Yes | No/Single vessel |

**Aerobic Training (Weeks 1-3):**

| Parameter | Initial | Progression |
|---|---|---|
| Mode | Treadmill walking, cycle ergometer | Add rowing, elliptical |
| Intensity | 40-50% HRR or RPE 11-12 | Progress to 50-60% HRR |
| Duration | 15-20 min | Increase by 2 min/session |
| Frequency | 3x/week | Progress to 3-5x/week |
| Warm-up/Cool-down | 5 min each | Essential components |

**HR Reserve Training Zones (Karvonen):**
```
Target HR = [(HRmax - HRrest) x %intensity] + HRrest

Example: 65-year-old, HRrest=65, target 50% HRR:
HRmax = 220 - 65 = 155
HRR = 155 - 65 = 90
Target HR = (90 x 0.50) + 65 = 110 bpm
```

**Heart Rate Training Zones by Risk Level:**

| Risk Level | Initial HR Target (% HRR) | Initial RPE | Progression |
|---|---|---|---|
| Low | 50-70% | 11-13 | Up to 85% HRR |
| Moderate | 40-60% | 11-12 | Up to 75% HRR |
| High | 30-50% | 10-11 | Up to 65% HRR, telemetry |

**Initial Resistance Training (Week 2-3, if cleared):**

| Parameter | Prescription |
|---|---|
| Frequency | 2x/week (non-consecutive days) |
| Intensity | 30-40% 1RM or very light bands |
| Sets x Reps | 1 set x 10-15 reps initially |
| Exercises | 6-8 exercises (major muscle groups) |
| Breathing | Exhale on exertion (no Valsalva) |
| Rest between sets | 1-2 minutes |
| Progression | Add second set in week 3 if tolerated |

**Resistance Exercise Selection:**

| Exercise | Muscle Groups | Notes |
|---|---|---|
| Seated chest press | Pectorals, triceps | Start with very light resistance |
| Seated row | Rhomboids, lats, biceps | Proper posture |
| Leg press | Quadriceps, gluteals | Monitor BP response |
| Calf raises | Gastrocnemius, soleus | Can perform bilaterally |
| Bicep curls | Biceps | Light dumbbells |
| Tricep extension | Triceps | Light resistance |
| Shoulder press | Deltoids | Start below shoulder height if needed |
| Abdominal exercises | Core | Avoid straining (no Valsalva) |

**Education Components:**

| Topic | Content | Format |
|---|---|---|
| Anatomy & disease | Heart function, disease process | Lecture/discussion |
| Risk factors | Modifiable vs. non-modifiable | Individual counseling |
| Nutrition | Heart-healthy diet, sodium, fat | Dietitian consultation |
| Medications | Purpose, side effects, adherence | Pharmacist consultation |
| Physical activity | Home exercise prescription | Individual plan |
| Psychosocial | Stress management, depression | Screening + referral |
| Smoking cessation | If applicable | Counseling + pharmacotherapy |
| Return to work/driving | Guidelines, timing | Individual assessment |

**Symptom Monitoring (Stop Exercise If):**
- Chest pain/angina (radiating to arm, jaw, back)
- Excessive dyspnea (unable to speak)
- Dizziness, lightheadedness, or confusion
- Palpitations or irregular heartbeat
- Leg pain (claudication)
- Excessive fatigue (RPE >15)
- Signs of poor perfusion (pale, clammy)
- Systolic BP >250 or <90 mmHg
- Diastolic BP >115 mmHg
- Heart rate exceeds prescribed limit

## 5. Integration with Neuromodulation

Neuromodulation techniques, when combined with rehabilitation exercises, can enhance neuroplasticity and accelerate motor recovery. This section details evidence-based combinations, timing protocols, and clinical implementation.

### 5.1 Overview of Neuromodulation Modalities

| Modality | Mechanism | Application | Evidence Level |
|---|---|---|---|
| tDCS | Cathodal/anodal polarization of cortex | Stroke UE, pain, cognition | Moderate-Strong |
| tACS | Oscillatory entrainment | Motor learning, tremor | Emerging |
| tRNS | Stochastic resonance, noise-enhanced signaling | Motor learning, balance | Moderate |
| TMS | Induced action potentials, LTP/LTD | Depression, motor recovery, aphasia | Strong |
| PBM/LLLT | Mitochondrial stimulation, reduced oxidative stress | Pain, tissue healing, muscle recovery | Moderate |
| Peripheral NES | Sensory/motor nerve stimulation | Spasticity, muscle activation, pain | Strong |
| VNS (non-invasive) | Locus coeruleus activation | Stroke, tinnitus | Emerging |
| ECoG | Direct cortical recording/stimulation | Research/implanted | Research |

### 5.2 tDCS + Motor Training (Dual Approach)

**Mechanism:**
tDCS modulates cortical excitability through subthreshold polarization:
- **Anodal tDCS:** Increases cortical excitability (depolarization) → Facilitates motor learning
- **Cathodal tDCS:** Decreases cortical excitability (hyperpolarization) → Suppresses maladaptive activity (e.g., contralesional inhibition in stroke)

**Parameters:**

| Parameter | Standard Protocol | Variations |
|---|---|---|
| Current intensity | 1-2 mA | Up to 4 mA (investigational) |
| Electrode size | 25-35 cm squared | Smaller for focal stimulation |
| Duration | 20 minutes | 10-30 minutes reported |
| Montage | Specific to target (see below) | Bipolar, HD-tDCS |
| Session frequency | 5-10 sessions (consecutive or weekly) | Up to 20+ sessions |

**tDCS Montages for Motor Rehabilitation:**

| Condition | Target | Anode Placement | Cathode Placement |
|---|---|---|---|
| Stroke UE (ipsilesional M1) | Increase ipsilesional excitability | Ipsilesional M1 (C3/C4) | Contralesional M1 (C4/C3) |
| Stroke UE (contralesional inhibition) | Decrease contralesional inhibition | Ipsilesional M1 | Contralesional M1 |
| Stroke LE | Increase LE cortical representation | Ipsilesional leg M1 (Cz-1cm) | Contralesional leg M1 |
| Chronic pain | M1 stimulation | Contralateral M1 to pain | Contralateral orbitofrontal |
| Cognitive (attention) | Left DLPFC | F3 | Right supraorbital |
| Cognitive (executive) | Bilateral DLPFC | F3/F4 alternating | Right/left supraorbital |

**tDCS + Exercise Protocol (Stroke Upper Extremity Example):**

| Step | Activity | Duration |
|---|---|---|
| 1 | Setup: Shave/clean skin, position electrodes per montage | 5 min |
| 2 | Begin tDCS: Ramp up to target current (1-2 mA over 30s) | 30s ramp-up |
| 3 | During tDCS: Begin motor training (RTP, CIMT, robotics) | 20 min concurrent |
| 4 | tDCS ramp-down | 30s ramp-down |
| 5 | Continue motor training (without tDCS) | 20-40 min |
| 6 | Session total | 45-65 min |

**Evidence Summary: tDCS + Motor Training:**

| Study/Review | Population | tDCS Protocol | Training | Outcomes |
|---|---|---|---|---|
| Lefebvre et al. (2015) | Subacute stroke | 2 mA, 20 min, ipsilesional M1 | 30 min OT | FMA-UE improvement: +3.5 vs +1.5 (sham) |
| Stagg et al. (2012) | Chronic stroke | 1 mA, bilateral M1 | Constraint-induced | FMA improvement greater than sham |
| Elsner et al. (2016) | Meta-analysis (Stroke) | Various | Various | Small to moderate effect on ADL (SMD 0.24) |
| Marquez et al. (2013) | Chronic stroke | 1 mA, 20 min, M1 | 45 min task training | FMA-UE: significant improvement over sham |
| Grecco et al. (2014) | Children with CP | 1 mA, M1 | Gait training | GMFM improvement with tDCS |

**Key Evidence-Based Recommendations:**
1. Concurrent tDCS + motor training is more effective than sequential or either alone
2. Multiple sessions (5-10) produce more durable effects than single session
3. 1-2 mA appears to be the optimal intensity range
4. Effects are cumulative with repeated sessions
5. Individual response varies significantly (responder vs. non-responder phenomenon)

**Practical Implementation Protocol:**

```
SESSION STRUCTURE (Example: Post-stroke upper extremity)
============================================================
Pre-session:
  - Screen for contraindications (see Section 7)
  - Assess baseline motor function (FMA-UE, WMFT)
  - Prepare skin (cleanse, minimal impedance <10k ohms)
  - Position electrodes (Anode: ipsilesional C3/C4, Cathode: contralesional C4/C3)

tDCS Phase (20 minutes):
  - RAMP UP: 0 to 2 mA over 30 seconds
  - STEADY: 2 mA maintained for 20 minutes
  - Begin motor training at minute 0 (concurrent stimulation)
  - Training: Repetitive task practice, 300+ repetitions
  - MONITOR: Skin sensation, comfort, adverse events
  - RAMP DOWN: 2 mA to 0 over 30 seconds

Post-tDCS Phase (20-40 minutes):
  - Continue motor training without stimulation
  - Functional task practice
  - Progress exercise difficulty

Post-session:
  - Assess skin condition under electrodes
  - Document session parameters
  - Record subjective response
  - Schedule next session (preferably consecutive days)

COURSE: 10 sessions (2 weeks: 5 sessions/week, or 10 sessions over 3-4 weeks)
FOLLOW-UP: Reassess motor function at end of course, 1 month, 3 months
```

### 5.3 TMS + Constraint-Induced Therapy

**Mechanism:**
TMS uses electromagnetic induction to generate action potentials in cortical neurons:
- **rTMS (repetitive TMS):** Trains of pulses at specific frequencies
  - High-frequency (>5 Hz): Facilitates cortical excitability (LTP-like)
  - Low-frequency (<=1 Hz): Suppresses cortical excitability (LTD-like)
- **TBS (Theta-Burst Stimulation):** Patterned stimulation mimicking hippocampal theta rhythm
  - iTBS (intermittent): Facilitatory
  - cTBS (continuous): Inhibitory

**TMS Parameters:**

| Parameter | Standard | Notes |
|---|---|---|
| Coil type | Figure-of-8 (focal) or H-coil (deeper) | Figure-of-8 for M1 |
| Stimulation intensity | 90-120% resting motor threshold (RMT) | Individualized per patient |
| Frequency (rTMS) | 1 Hz (inhibitory) or 10 Hz (facilitatory) | Condition-dependent |
| Pulses per session | 600-3000 | Protocol-dependent |
| Duration | 10-30 minutes | Protocol-dependent |

**TMS + CIMT Protocol (Stroke Upper Extremity):**

| Phase | Timing | Protocol |
|---|---|---|
| 1. Motor threshold determination | Day 1 | Determine RMT for ipsilesional M1 |
| 2. Low-frequency rTMS to contralesional M1 | Before CIMT daily | 1 Hz, 20 min (1200 pulses), 90% RMT |
| 3. CIMT training | Immediately after TMS | 6 hours/day, 2 weeks |
| 4. Restraint | Continuous during waking | Unaffected limb restrained |
| 5. Assessment | Pre, mid, post, follow-up | FMA-UE, WMFT, MAL |

**TBS + Therapy Protocol:**

| Stimulation Type | Protocol | Timing |
|---|---|---|
| iTBS (facilitatory) to ipsilesional M1 | 600 pulses, before training | Pre-exercise, 5x/week |
| cTBS (inhibitory) to contralesional M1 | 600 pulses, before training | Pre-exercise, 5x/week |

**Evidence Summary: TMS + Rehabilitation:**

| Study/Review | Population | TMS Protocol | Rehabilitation | Key Findings |
|---|---|---|---|---|
| Hsu et al. (2012) | Stroke | LF-rTMS contralesional M1 | PT/OT | Greater motor improvement vs. sham + rehab |
| Lefaucheur et al. (2014) | Meta-analysis (Stroke) | LF-rTMS contra M1 | Various | Significant effect on motor recovery (Level A evidence) |
| Di Lazzaro et al. (2019) | Stroke | iTBS | Motor training | iTBS enhances motor learning |
| Ackerley et al. (2016) | Healthy + Stroke | cTMS | Motor training | Timing-dependent plasticity demonstrated |

**Clinical Implementation:**

```
TMS SESSION STRUCTURE
=====================
Pre-session:
  - Screen for TMS contraindications (metal implants, seizure history, medications)
  - Determine motor hotspot and resting motor threshold
  - Map optimal coil position using neuronavigation (if available)

TMS Protocol:
  - Position coil over target (e.g., contralesional M1)
  - Deliver low-frequency rTMS (1 Hz) OR iTBS/cTBS
  - Monitor for adverse effects (headache, discomfort, syncope)

Post-TMS:
  - Immediately begin rehabilitation session (within 15 minutes)
  - 30-60 minutes of motor training
  - Capitalize on TMS-induced metaplasticity window

Safety:
  - Seizure risk is low but present (<0.01% per session)
  - Ear protection required for acoustic artifact
  - Screen for metallic implants in head/neck
  - Staff training in seizure first aid
```

### 5.4 tRNS + Balance Training

tRNS (transcranial Random Noise Stimulation) applies random electrical oscillations (typically 100-640 Hz) to the cortex, thought to enhance neural signal processing through stochastic resonance.

**Mechanism:**
- Stochastic resonance: Adding optimal noise enhances weak signal detection
- Facilitates long-term potentiation
- May work through sodium channel modulation
- Less directional than tDCS (does not require specific polarity)

**tRNS Parameters for Balance Training:**

| Parameter | Standard Protocol |
|---|---|
| Frequency range | 100-640 Hz (broadband) |
| Current intensity | 1-1.5 mA (peak-to-peak) |
| Duration | 10-15 minutes |
| Electrode placement | Bilateral cerebellum OR Cz + extracephalic |
| Session frequency | 3-5x/week for 2-4 weeks |

**tRNS + Balance Training Protocol:**

| Step | Activity | Duration |
|---|---|---|
| 1 | tRNS setup (bilateral cerebellar montage) | 5 min |
| 2 | tRNS begins (1-1.5 mA) | 10 min |
| 3 | Concurrent balance exercises begin | 10 min during tRNS |
| 4 | Continue balance training post-tRNS | 20 min |
| 5 | Cool-down and assessment | 5 min |

**Evidence:**
- Cerebellar tRNS has shown improvement in balance and motor learning
- Less studied than tDCS but promising for balance/cerebellar functions
- May be particularly effective when combined with challenging balance tasks

### 5.5 PBM + Exercise Recovery

PBM (Photobiomodulation), also known as LLLT (Low-Level Laser Therapy), uses red/near-infrared light to stimulate cellular energy production and reduce inflammation.

**Mechanism:**
- Cytochrome c oxidase stimulation in mitochondria
- Increased ATP production

### 5.6 Timing: Pre, During, Post-Exercise Neuromodulation

**Timing Framework:**

| Timing | Stimulation | Exercise | Expected Effect | Evidence Strength |
|---|---|---|---|---|
| **Pre-exercise** (10-30 min before) | tDCS, TMS, tRNS | Motor training | Prime cortex for learning, enhance LTP | Strong |
| **Concurrent** (during exercise) | tDCS, tRNS | Motor, balance training | Reinforce active neural circuits, Hebbian learning | Strong |
| **Post-exercise** (immediately after) | tDCS, TMS | - | Consolidate motor memory, offline learning | Moderate |
| **Interleaved** (between sets) | tDCS, PBM | Resistance training | Maintain excitability, enhance recovery | Emerging |

**Optimal Timing by Modality:**

| Modality | Optimal Timing | Rationale |
|---|---|---|
| tDCS | Concurrent with training | Hebbian plasticity: neurons that fire together, wire together |
| TMS | Pre-training (within 15 min) | Metaplasticity window, lasting effects 15-60 min |
| tRNS | Concurrent or pre-training | Stochastic resonance during task performance |
| PBM | Pre and/or post-exercise | Preconditioning + recovery enhancement |

**Pre-During-Post Integration Protocol (Example: Stroke Upper Extremity with tDCS):**

```
FULL SESSION: tDCS + Motor Training + PBM
=============================================

PRE-SESSION (15 minutes):
  1. Patient setup, vitals check
  2. Skin preparation for tDCS electrodes
  3. Place tDCS electrodes (anode: ipsilesional M1, cathode: contralesional M1)
  4. PBM application to affected upper extremity muscles (60s per site)
     - Biceps brachii
     - Triceps brachii
     - Wrist flexors
     - Wrist extensors
     - Shoulder muscles

STIMULATION + TRAINING (45 minutes):
  0:00  - Begin tDCS (ramp to 2 mA)
  0:00  - Begin motor training block 1: Warm-up ROM (5 min)
  5:00  - Motor training block 2: Repetitive task practice (20 min)
          - 300-400 repetitions of functional tasks
          - Progressive difficulty
  25:00 - Motor training block 3: Functional integration (15 min)
          - Bimanual tasks
          - ADL simulation
  40:00 - Cool-down, stretching (5 min)
  40:00 - tDCS ramp-down complete

POST-SESSION (10 minutes):
  1. PBM application to trained muscles (post-exercise recovery)
     - 60s per site to exercised muscle groups
  2. Skin inspection under tDCS electrodes
  3. Outcome documentation
  4. Home exercise assignment

TOTAL SESSION TIME: ~70 minutes
RECOMMENDED FREQUENCY: 5x/week for 2-4 weeks
```

### 7.6 Pain Monitoring

**Exercise Pain Rules:**

| Pain Level | Response | Action |
|---|---|---|
| 0-2/10 | Acceptable during exercise | Continue as planned |
| 3-4/10 | Mild pain during exercise | Monitor, acceptable if resolves within 24 hours |
| 5/10 | Moderate pain | Modify exercise (reduce intensity/range), assess |
| 6-7/10 | Significant pain | Stop exercise, reassess, modify program |
| 8-10/10 | Severe pain | Stop immediately, assess for injury, refer if needed |

**Key Pain Principles:**

1. **Pain during exercise:** Some discomfort is normal with strengthening; sharp, shooting, or radiating pain is not
2. **Pain after exercise:** Mild soreness (DOMS) is normal 24-48 hours post new exercise; pain that progressively worsens is concerning
3. **Pain vs. exertion:** Teach patients difference between muscle exertion/burn (normal) and pain (potentially harmful)
4. **Chronic pain patients:** Use graded exposure, pain neuroscience education; pain does not always mean harm
5. **Centralization:** In radicular conditions, pain moving from distal to proximal = favorable response
6. **Peripheralization:** Pain moving from proximal to distal = unfavorable, modify treatment

**Pain Monitoring Tools:**

| Tool | Application | Scale |
|---|---|---|
| Visual Analog Scale (VAS) | General pain | 0-100 mm line |
| Numeric Rating Scale (NRS) | General pain | 0-10 |
| Verbal Rating Scale (VRS) | General pain | None, mild, moderate, severe |
| Borg CR-10 | Exertion + pain combined | 0-10 (special category for maximal pain) |
| Pain diary | Longitudinal tracking | Multiple entries per day |
| McGill Pain Questionnaire | Multidimensional assessment | Sensory, affective, evaluative |
| Pain Catastrophizing Scale | Psychological component | 13 items |
| Tampa Scale of Kinesiophobia | Fear of movement | 17 items |

### 7.10 Exercise Modification Hierarchy

When adverse events or contraindications arise, use this modification hierarchy:

```
MODIFICATION HIERARCHY (least to most restrictive)
====================================================

Step 1: MODIFY PARAMETERS
  - Reduce intensity (lower % 1RM, lower HR target)
  - Reduce volume (fewer sets/reps, shorter duration)
  - Reduce frequency (fewer sessions per week)
  - Increase rest periods
  - Change exercise mode (e.g., land to water)

Step 2: MODIFY TECHNIQUE
  - Adjust range of motion
  - Change body position (e.g., standing to seated)
  - Add support/props
  - Use assistive devices
  - Slow down tempo

Step 3: SUBSTITUTE EXERCISES
  - Choose alternative with similar benefits
  - Reduce weight-bearing component
  - Use machine instead of free weights
  - Choose non-impact options

Step 4: DISCONTINUE SPECIFIC EXERCISE
  - Stop the problematic exercise only
  - Continue other exercises in program
  - Reintroduce later if appropriate

Step 5: DISCONTINUE SESSION
  - Stop current exercise session
  - Assess and address the issue
  - Resume at next scheduled session (possibly modified)

Step 6: DISCONTINUE PROGRAM
  - Medical referral required
  - Reassess before resuming
  - Consider alternative interventions
```

### 8.2 Goal Attainment Scaling (GAS)

GAS is a criterion-referenced, individualized outcome measure that quantifies goal achievement on a standardized scale.

#### 8.2.1 GAS Methodology

**Scaling Framework:**

| Score | Level | Description |
|---|---|---|
| -2 | Much less than expected | Patient achieves much less than expected (significant deterioration or no progress) |
| -1 | Somewhat less than expected | Patient achieves somewhat less than expected (partial progress) |
| 0 | Expected outcome | Patient achieves expected level (primary goal met) |
| +1 | Somewhat more than expected | Patient achieves somewhat more than expected (better than expected) |
| +2 | Much more than expected | Patient achieves much more than expected (exceptional outcome) |

**GAS Goal-Setting Process:**

1. **Collaborative goal identification:** Patient + therapist + family identify 3-5 meaningful goals
2. **Operationalize goals:** Define observable, measurable behaviors/outcomes
3. **Determine expected outcome (score = 0):** Realistic expected level at follow-up
4. **Define other scale levels:** Describe what -2, -1, +1, +2 would look like for each goal
5. **Assign goal weights (optional):** Weight goals by importance to patient (1-3 scale)

**GAS Score Calculation:**

```
Individual goal score: -2 to +2

Composite GAS Score (T-score):
T = 50 + [10 x (sum of goal scores) / sqrt(number of goals)]

Example for 4 goals with scores: +1, 0, +1, -1
Sum = +1
T = 50 + [10 x 1 / sqrt(4)]
T = 50 + [10 x 1 / 2]
T = 50 + 5 = 55

Interpretation:
T = 50: Exactly at expected level
T > 50: Better than expected
T < 50: Worse than expected
```

#### 8.2.2 GAS Example for Post-Stroke Patient

| Goal | -2 | -1 | 0 (Expected) | +1 | +2 |
|---|---|---|---|---|---|
| **Feeding** | Requires full assistance | Requires setup + moderate assist | Independent with setup only | Independent, no setup | Independent, can prepare food |
| **Walking** | Unable to walk with walker | Walks 10m with walker + assist | Walks 50m with walker independently | Walks 100m with walker | Walks with cane or unassisted |
| **Dressing upper body** | Requires full assistance | Requires moderate assistance | Independent with adapted clothing | Independent with all clothing | Dresses including buttons/zippers |
| **Transfer bed-chair** | Requires 2-person assist | Requires 1-person assist + equipment | Independent with equipment | Independent, no equipment | Independent, including car transfer |

**GAS Advantages:**
- Patient-centered and individualized
- Sensitive to small but meaningful changes
- Captures multidimensional outcomes
- Applicable across diverse populations
- Engages patient in goal-setting
- Valid for group comparisons despite individualization (via T-scores)

**GAS Limitations:**
- Requires training to set reliable goals
- Time-intensive to establish goals properly
- Risk of setting too easy goals (score inflation)
- Inter-rater reliability varies
- Ceiling effects possible if goals set too low

#### 8.3.2 MCID vs. MDC vs. SEM

| Concept | Definition | Application |
|---|---|---|
| **SEM** (Standard Error of Measurement) | Precision of the measure | Individual score reliability |
| **MDC** (Minimal Detectable Change) | Smallest change > measurement error | Determines if change is "real" |
| **MCID** (Minimal Clinically Important Difference) | Smallest meaningful change to patient | Determines if change matters |
| **SRD** (Smallest Real Difference) | Same as MDC | Research applications |

```
Relationship:
MCID >= MDC (ideally): Change that is both real AND meaningful
If MCID < MDC: Instrument may not be sensitive enough

MDC95 = 1.96 x SEM x sqrt(2)
SEM = SD x sqrt(1 - reliability)
```

#### 8.3.3 Distribution-Based Methods

| Method | Formula | Interpretation |
|---|---|---|
| Effect size | Mean change / SD baseline | Small: 0.2, Medium: 0.5, Large: 0.8 |
| Standardized response mean | Mean change / SD of change | Small: 0.2, Medium: 0.5, Large: 0.8 |
| Minimal detectable change | 1.96 x SEM x sqrt(2) | Change > this is not due to error |

#### 8.5.3 Quality Improvement Framework

```
PLAN-DO-STUDY-ACT (PDSA) CYCLE FOR REHAB OUTCOMES
===================================================

PLAN:
1. Identify outcome measure to improve (e.g., 6MWT in COPD)
2. Establish baseline performance (current MCID attainment rate)
3. Set target (e.g., increase MCID attainment from 55% to 70%)
4. Identify potential interventions:
   - Increase exercise intensity
   - Implement interval training
   - Add respiratory muscle training
   - Improve adherence strategies
5. Plan implementation: who, what, when, where

DO:
1. Implement intervention with defined population
2. Collect data prospectively
3. Document process and unexpected observations
4. Run for defined period (e.g., 3 months)

STUDY:
1. Compare outcomes to baseline
2. Analyze data: Did MCID attainment improve?
3. Identify barriers and facilitators
4. Determine if intervention was effective

ACT:
1. If effective: Standardize the change, implement broadly
2. If partially effective: Modify and run new PDSA cycle
3. If not effective: Try different intervention
4. Continue monitoring over time
```

### 8.6 Psychometric Properties Reference

When selecting outcome measures, consider these psychometric properties:

| Property | Definition | Acceptable Threshold |
|---|---|---|
| **Reliability** | Consistency of measurement | |
| Test-retest reliability | Consistency over time | ICC >0.75 |
| Inter-rater reliability | Consistency between raters | ICC >0.75 or Kappa >0.6 |
| Intra-rater reliability | Consistency within same rater | ICC >0.80 |
| Internal consistency | Items measure same construct | Cronbach's alpha 0.70-0.95 |
| **Validity** | Measures what it intends to measure | |
| Content validity | Covers all aspects of construct | Expert panel review |
| Criterion validity | Correlates with gold standard | r >0.70 with reference |
| Construct validity | Theoretically expected relationships | Hypothesis testing confirmed |
| **Responsiveness** | Detects change over time | |
| Effect size | Mean change / SD baseline | >0.2 small, >0.5 medium, >0.8 large |
| Standardized response mean | Mean change / SD of change | >0.2 small, >0.5 medium, >0.8 large |
| Area under ROC curve | Ability to discriminate change | >0.70 |
| **Interpretability** | Meaning of scores is clear | |
| Floor/ceiling effects | | <15% at extremes |
| MCID established | Clinically meaningful change | Published anchor/distribution method |
| Normative data available | Reference values | Population-representative |

**Treadmill Metabolic Equations (ACSM):**

```
Walking (3-5 mph):
VO2 = (0.1 x speed m/min) + (1.8 x speed x grade) + 3.5 ml/kg/min

Running (>5 mph):
VO2 = (0.2 x speed m/min) + (0.9 x speed x grade) + 3.5 ml/kg/min

Leg cycling:
VO2 = (1.8 x work rate kg/m/min) / body weight kg + 7 ml/kg/min

Arm cycling:
VO2 = (3 x work rate kg/m/min) / body weight kg + 3.5 ml/kg/min

Stepping:
VO2 = (0.2 x steps/min) + (1.33 x 1.8 x step height m x steps/min) + 3.5
```
