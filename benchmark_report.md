# DeepSynaps Studio Clinical AI Benchmark Report

## Benchmark basis

This audit compared the repository implementation against current market and research patterns: MNE-Python/PyPREP/autoreject/ICLabel/SpecParam/qEEG report workflows; MRIQC/BIDS/NIfTI-native neuroimaging QC, volumetrics, brain-age and longitudinal MRI pipelines; multimodal clinical decision-support practices emphasizing source verification, provenance, uncertainty, and clinician review.

Scores use 10 dimensions: technical pipeline quality, clinical relevance, explainability, confidence/uncertainty, robustness, multimodal readiness, reporting quality, deployability, API architecture, and UI/workflow fit.

## Deferred high-impact upgrades

1. Patient-specific EEG source localization with MRI coregistration and BIDS derivatives.
2. Full MRI normative percentiles using LMS/GAMLSS age/sex references and calibrated external validation.
3. Learned multimodal models with conformal prediction after sufficient paired qEEG+MRI+assessment outcomes.
4. Evidence quality grading with counter-evidence and patient-applicability scoring for every generated claim.
5. Clinician validation studies and report usability testing before stronger clinical claims.
