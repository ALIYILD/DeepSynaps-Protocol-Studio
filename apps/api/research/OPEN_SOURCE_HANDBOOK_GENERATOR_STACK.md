### #1 — python-docx-template (docxtpl)
**Category:** DOCX Template Engine with Jinja2

| Attribute | Detail |
|-----------|--------|
| **GitHub URL** | https://github.com/elapouya/python-docx-template |
| **License** | LGPL-2.1-only |
| **Stars** | ~2,547 |
| **Last Updated** | Active (3 months ago at time of research) |
| **PyPI** | `pip install docxtpl` |

**Key Features:**
- Use Microsoft Word .docx files as Jinja2 templates
- Supports rich templating: `{{ variables }}`, `{% for %}`, `{% if %}` blocks
- Inline images, tables, and paragraph manipulation within templates
- Sub-document composition (merge multiple DOCX files)
- Preserves Word formatting, styles, headers/footers
- Integration with `docxcompose` for multi-document merging

**DeepSynaps Integration Recommendation:**
- **PRIMARY RECOMMENDATION** for DOCX clinical document generation
- Create clinical templates (discharge summaries, SOPs, patient education) as .docx files with Jinja2 placeholders
- DeepSynaps API generates documents by rendering templates with patient/protocol data
- Perfect for regulatory documents that must be in DOCX format

**Code Example:**
```python
from docxtpl import DocxTemplate

# Load a clinical template (e.g., discharge_summary_template.docx)
doc = DocxTemplate("clinical_templates/discharge_summary.docx")

# Context data from DeepSynaps
context = {
    "patient_name": "Jane Doe",
    "mrn": "MRN-2025-001234",
    "date_of_birth": "1985-03-15",
    "admission_date": "2025-08-01",
    "discharge_date": "2025-08-07",
    "attending_physician": "Dr. Sarah Chen",
    "primary_diagnosis": "Community Acquired Pneumonia",
    "medications": [
        {"name": "Amoxicillin 500mg", "frequency": "TID x 7 days"},
        {"name": "Albuterol inhaler", "frequency": "PRN"},
    ],
    "follow_up_instructions": "Follow up with PCP in 1 week. Return if fever > 101F.",
    "allergies": ["Penicillin", "Sulfa"]
}

# Render and save
doc.render(context)
doc.save("output/discharge_summary_jane_doe.docx")
```

**Key Features:**

- Tables, multi-cell text, automatic page breaks
- HTML to PDF conversion (`write_html()`)

- Harvard CS50 uses this library

**DeepSynaps Integration Recommendation:**
- **PRIMARY RECOMMENDATION** for PDF clinical report generation
- Generate patient reports, lab results, clinical summaries as PDF
- Use Jinja2 + HTML templates → PDF pipeline for maximum flexibility
- Ideal for secure, tamper-evident document generation
- Can embed digital signatures for document authenticity

**Code Example:**
```python
from fpdf import FPDF

class ClinicalPDF(FPDF):
    def header(self):
        self.set_font("Helvetica", "B", 14)
        self.cell(0, 10, "DeepSynaps Clinical Report", ln=True, align="C")
        self.set_font("Helvetica", "", 10)
        self.cell(0, 6, "Generated by DeepSynaps Protocol Studio", ln=True, align="C")
        self.ln(5)
        # Draw a line
        self.set_draw_color(0, 102, 204)
        self.line(10, self.get_y(), 200, self.get_y())
        self.ln(5)

    def footer(self):
        self.set_y(-15)
        self.set_font("Helvetica", "I", 8)
        self.set_text_color(128)
        self.cell(0, 10, f"Page {self.page_no()} | Confidential - Authorized Use Only", align="C")

# Generate report
pdf = ClinicalPDF()
pdf.add_page()
pdf.set_auto_page_break(auto=True, margin=20)

# Patient header
pdf.set_font("Helvetica", "B", 12)
pdf.cell(0, 8, "Patient: Jane Doe | MRN: MRN-2025-001234", ln=True)
pdf.set_font("Helvetica", "", 10)
pdf.cell(0, 6, "DOB: 1985-03-15 | Attending: Dr. Sarah Chen", ln=True)
pdf.ln(5)

# Clinical notes
pdf.set_font("Helvetica", "B", 11)
pdf.cell(0, 8, "Clinical Assessment", ln=True)
pdf.set_font("Helvetica", "", 10)
assessment = """The patient presented with acute onset dyspnea and fever. Chest X-ray revealed bilateral infiltrates consistent with community acquired pneumonia. Patient was started on empiric antibiotic therapy with Amoxicillin. Clinical improvement noted with resolution of fever within 48 hours."""
pdf.multi_cell(0, 6, assessment)
pdf.ln(5)

# Medications table
pdf.set_font("Helvetica", "B", 11)
pdf.cell(0, 8, "Medications", ln=True)
pdf.set_font("Helvetica", "", 10)

# Simple table
with pdf.table() as table:
    # Header
    row = table.row()
    row.cell("Medication", style=pdf.font_style_b)
    row.cell("Dose")
    row.cell("Frequency")
    row.cell("Duration")
    # Data
    row = table.row()
    row.cell("Amoxicillin")
    row.cell("500mg")
    row.cell("TID")
    row.cell("7 days")
    row = table.row()
    row.cell("Albuterol")
    row.cell("90mcg inhaler")
    row.cell("PRN")
    row.cell("As needed")

pdf.output("clinical_report.pdf")
```

**Key Features:**

- Document properties (author, title, subject, keywords)
- Relationship management (hyperlinks, embedded objects)

**DeepSynaps Integration Recommendation:**
- **Foundation library** — used by docxtpl, markdown2docx, and others
- Use directly for simple document creation without templating
- Combine with Jinja2 manually for lightweight templating
- Essential dependency for most DOCX workflows

**Code Example:**
```python
from docx import Document
from docx.shared import Inches, Pt, RGBColor
from docx.enum.text import WD_ALIGN_PARAGRAPH

# Create clinical document
doc = Document()

# Title
title = doc.add_heading("Patient Care Plan", level=0)
title.alignment = WD_ALIGN_PARAGRAPH.CENTER

# Patient info table
table = doc.add_table(rows=4, cols=2)
table.style = "Light Grid Accent 1"
table.cell(0, 0).text = "Patient Name"
table.cell(0, 1).text = "Jane Doe"
table.cell(1, 0).text = "MRN"
table.cell(1, 1).text = "MRN-2025-001234"
table.cell(2, 0).text = "Date of Birth"
table.cell(2, 1).text = "1985-03-15"
table.cell(3, 0).text = "Attending Physician"
table.cell(3, 1).text = "Dr. Sarah Chen"
doc.add_paragraph()  # Spacing

# Assessment
doc.add_heading("Assessment & Plan", level=1)
doc.add_paragraph("Diagnosis: Community Acquired Pneumonia")
p = doc.add_paragraph()
p.add_run("Severity: ").bold = True
p.add_run("Moderate")

# Plan items
plan = doc.add_paragraph()
plan.add_run("Plan:").bold = True
doc.add_paragraph("1. Complete antibiotic course (Amoxicillin 500mg TID x 7 days)", style="List Number")
doc.add_paragraph("2. Follow-up chest X-ray in 4 weeks", style="List Number")
doc.add_paragraph("3. PCP follow-up within 7 days", style="List Number")
doc.add_paragraph("4. Return precautions: Fever >101F, worsening dyspnea, hemoptysis", style="List Number")

# Add styled note
note = doc.add_paragraph()
run = note.add_run("IMPORTANT: Patient has documented allergy to Penicillin. Verify antibiotic selection.")
run.font.color.rgb = RGBColor(255, 0, 0)
run.font.bold = True
run.font.size = Pt(10)

# Footer
section = doc.sections[0]
footer = section.footer
footer_para = footer.paragraphs[0] if footer.paragraphs else footer.add_paragraph()
footer_para.text = "DeepSynaps Protocol Studio | Confidential | Page "
footer_para.alignment = WD_ALIGN_PARAGRAPH.CENTER

doc.save("patient_care_plan.docx")
```

    "clinical_report_weasy.pdf",

)

from borb.pdf.canvas.layout.text.heading import Heading
from borb.pdf.canvas.color import HexColor

# Create document
document = Document()
page = Page()
document.append_page(page)
layout = SingleColumnLayout(page)

# Add content
layout.add(Heading("DeepSynaps Clinical Report", level=1))
layout.add(Paragraph("Patient: Jane Doe | MRN: MRN-2025-001234"))
layout.add(Paragraph("DOB: 1985-03-15 | Attending: Dr. Sarah Chen"))
layout.add(Paragraph(" "))  # Spacer

# Add diagnosis section
layout.add(Heading("Diagnosis", level=2))
layout.add(Paragraph("Primary: Community Acquired Pneumonia"))

# Add medications table
table = Table(number_of_rows=3, number_of_columns=3)
table.add(TableCell(Paragraph("Medication"), background_color=HexColor("0056b3")))
table.add(TableCell(Paragraph("Dose"), background_color=HexColor("0056b3")))
table.add(TableCell(Paragraph("Frequency"), background_color=HexColor("0056b3")))
table.add(TableCell(Paragraph("Amoxicillin")))
table.add(TableCell(Paragraph("500mg")))
table.add(TableCell(Paragraph("TID x 7 days")))
table.add(TableCell(Paragraph("Albuterol")))
table.add(TableCell(Paragraph("90mcg")))
table.add(TableCell(Paragraph("PRN")))
layout.add(table)

# Save
with open("clinical_report_borb.pdf", "wb") as f:
    PDF.dumps(f, document)

### Security Considerations

1. **Input Sanitization**: All document processing pipelines must sanitize inputs — especially when processing uploaded DOCX/PDF files from external sources
2. **XXE Prevention**: DOCX files are ZIP/XML — guard against XML External Entity attacks when parsing
3. **Memory Management**: Large document batch jobs should use streaming/concurrent processing to avoid memory exhaustion
4. **Data Privacy**: Generated clinical documents must not be logged or stored outside approved data stores
5. **License Compliance**: AGPL tools (borb, rinohtype) require careful license management for commercial use
